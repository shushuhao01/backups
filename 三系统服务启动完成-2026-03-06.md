## 常见问题处理

### Q1: 端口被占用怎么办?

**查找占用进程**:
```bash
netstat -ano | findstr :端口号
```

**终止进程**:
```bash
taskkill /F /PID 进程ID
```

### Q2: 后端服务无法启动?

**检查日志**:
```bash
pm2 logs backend
```

**重新构建**:
```bash
cd backend
npm run build
pm2 restart backend
```

### Q3: 前端无法连接后端?

**检查后端状态**:
```bash
pm2 status
```

**检查端口**:
```bash
netstat -ano | findstr :3000
```

**测试API**:
```bash
curl http://localhost:3000/health
```

## 开发建议

### 1. 使用PM2管理所有服务

可以创建 `ecosystem.config.js` 统一管理:

```javascript
module.exports = {
  apps: [
    {
      name: 'backend',
      script: 'dist/app.js',
      cwd: './backend'
    },
    {
      name: 'crm-frontend',
      script: 'npm',
      args: 'run dev',
      cwd: './'
    },
    {
      name: 'website-frontend',
      script: 'npm',
      args: 'run dev',
      cwd: './website'
    },
    {
      name: 'admin-frontend',
      script: 'npm',
      args: 'run dev',
      cwd: './admin'
    }
  ]
};
```

然后使用:
```bash
pm2 start ecosystem.config.js
pm2 stop all
pm2 restart all
```

### 2. 使用VS Code任务

创建 `.vscode/tasks.json`:
```json
{
  "version": "2.0.0",
  "tasks": [
    {
      "label": "Start All Services",
      "dependsOn": [
        "Start Backend",
        "Start CRM",
        "Start Website",
        "Start Admin"
      ]
    }
  ]
}
```
