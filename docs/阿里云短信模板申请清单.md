# 阿里云短信模板申请清单

## 📋 模板申请说明

本文档包含云客CRM系统所需的所有阿里云短信模板。请按照以下清单逐条在阿里云短信服务控制台申请模板。

### 申请注意事项：
1. 阿里云短信变量格式：`${变量名}`
2. 验证码类模板需要选择"验证码"类型
3. 通知类模板需要选择"短信通知"类型
4. 模板内容需要符合阿里云审核规范，不能包含营销推广内容
5. 建议先申请签名，再申请模板
6. 每个模板申请后会获得一个模板CODE，需要记录下来配置到系统中

---

## 1️⃣ 注册验证码模板

**模板名称：** 注册验证码

**申请类型：** 验证码

**模板内容：**
```
您的注册验证码是：${code}，5分钟内有效，请勿泄露给他人。
```

**变量说明：**
- `code`: 6位数字验证码

**使用场景：** 
- 官网注册时发送手机验证码
- 用于验证用户手机号真实性

**对应文件：** 
- `backend/src/controllers/public/RegisterController.ts`
- `backend/src/services/VerificationCodeService.ts`

## 2️⃣ 账号开通通知（免费试用）

**模板名称：** 账号开通通知

**申请类型：** 短信通知

**模板内容：**
```
尊敬的${tenantName}，您的云客CRM账号已开通！租户编码：${tenantCode}，授权码：${licenseKey}，管理员账号为您的手机号，初始密码：${adminPassword}，套餐：${packageName}，到期日期：${expireDate}。请妥善保管账号信息，登录后请立即修改密码。
```

**变量说明：**
- `tenantName`: 租户名称（企业名称）
- `tenantCode`: 租户编码（短代码）
- `licenseKey`: 授权码（完整授权码）
- `adminPassword`: 初始密码
- `packageName`: 套餐名称
- `expireDate`: 到期日期（格式：2026-12-31）

**重要修改说明：**
- ❌ 已移除 `adminUsername` 变量，改为固定文字"管理员账号为您的手机号"（阿里云不允许手机号作为变量）
- ❌ 已移除 `loginUrl` 变量（阿里云不允许URL作为变量传入）
- ✅ 移除了开头的【】签名符号，签名会自动添加

**使用场景：** 
- 用户选择免费试用套餐完成注册后立即发送
- 包含登录所需的全部信息

**对应文件：** 
- `backend/src/controllers/public/RegisterController.ts` (register方法，仅免费套餐)

## 🔧 配置说明

### 1. 申请流程
1. 登录阿里云控制台
2. 进入"短信服务"
3. 先申请短信签名（如：【云客CRM】）
4. 再逐条申请短信模板
5. 等待审核（通常1-2个工作日）
6. 审核通过后获得模板CODE

### 2. 系统配置
申请通过后，需要在系统中配置：

**方式一：环境变量配置**
```env
ALIYUN_SMS_ACCESS_KEY_ID=your_access_key_id
ALIYUN_SMS_ACCESS_KEY_SECRET=your_access_key_secret
ALIYUN_SMS_SIGN_NAME=云客CRM
ALIYUN_SMS_TEMPLATE_CODE_VERIFY=SMS_xxxxxx  # 验证码模板CODE
ALIYUN_SMS_TEMPLATE_CODE_REGISTER=SMS_xxxxxx  # 注册成功模板CODE
ALIYUN_SMS_TEMPLATE_CODE_PAYMENT=SMS_xxxxxx  # 支付成功模板CODE
# ... 其他模板CODE
```

**方式二：Admin后台配置**
- 登录Admin后台
- 进入"系统设置" -> "通知服务配置"
- 配置阿里云短信参数和模板CODE

### 3. 模板CODE映射
在 `backend/src/services/AliyunSmsService.ts` 中维护模板CODE映射：

```typescript
const SMS_TEMPLATES = {
  VERIFY_CODE: 'SMS_xxxxxx',        // 验证码
  REGISTER_SUCCESS: 'SMS_xxxxxx',   // 注册成功
  PAYMENT_SUCCESS: 'SMS_xxxxxx',    // 支付成功
  RENEW_SUCCESS: 'SMS_xxxxxx',      // 续期成功
  PACKAGE_CHANGE: 'SMS_xxxxxx',     // 套餐变更
  QUOTA_CHANGE: 'SMS_xxxxxx',       // 配额变更
  ACCOUNT_SUSPEND: 'SMS_xxxxxx',    // 账号暂停
  ACCOUNT_RESUME: 'SMS_xxxxxx',     // 账号激活
  ACCOUNT_CANCEL: 'SMS_xxxxxx',     // 账号注销
  REFUND_SUCCESS: 'SMS_xxxxxx',     // 退款成功
  EXPIRE_REMIND: 'SMS_xxxxxx',      // 到期提醒
  EXPIRED_NOTICE: 'SMS_xxxxxx'      // 过期通知
}
```

## ⚠️ 注意事项

1. **字数限制**：阿里云短信单条最多500字符，建议控制在300字符以内
2. **变量规范**：变量名只能包含字母、数字、下划线
3. **签名要求**：签名需要与企业资质一致，需要提供营业执照等证明材料
4. **审核时间**：工作日通常1-2天，节假日可能延长
5. **测试环境**：开发环境可以使用固定验证码（123456）进行测试
6. **发送频率**：同一手机号1分钟内只能发送1条，1小时内最多5条，1天内最多10条
7. **费用说明**：验证码短信约0.045元/条，通知短信约0.045元/条
