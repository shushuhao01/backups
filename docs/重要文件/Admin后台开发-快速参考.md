# Admin后台开发 - 快速参考

## 📊 一句话总结
Admin后台是**平台管理系统**,用于管理私有部署授权和SaaS租户,**不交付给客户**,保护商业机密。

## 🎯 三大系统关系
```
Admin后台 ←→ CRM系统 ←→ 官网
   ↓           ↓         ↓
授权管理    授权验证   套餐展示
租户管理    租户登录   在线支付
版本发布    版本更新   注册账号
```

## 🔗 关键对接接口

### CRM系统调用Admin
```
POST /api/v1/license/activate        # 私有部署激活
GET  /api/v1/license/verify          # 私有部署验证
POST /api/v1/tenant-license/verify   # 租户授权验证
POST /api/v1/tenant-license/heartbeat # 租户心跳检测
```

### 官网调用Admin
```
GET  /api/v1/packages                # 获取套餐列表
POST /api/v1/payment/orders          # 创建支付订单
GET  /api/v1/payment/orders/:id/status # 查询订单状态
```
