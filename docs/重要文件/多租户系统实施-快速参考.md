## 🔑 关键代码

### 数据库迁移
```sql
-- 添加tenant_id字段
ALTER TABLE `users` 
  ADD COLUMN `tenant_id` VARCHAR(36) NULL AFTER `id`,
  ADD INDEX `idx_tenant_id` (`tenant_id`);

-- 修改唯一索引
ALTER TABLE `users` DROP INDEX `username`;
ALTER TABLE `users` ADD UNIQUE KEY `uk_tenant_username` (`tenant_id`, `username`);
```

### 租户认证
```typescript
// 租户认证中间件
export const tenantAuth = async (req, res, next) => {
  const token = req.headers.authorization?.replace('Bearer ', '')
  const decoded = jwt.verify(token, process.env.JWT_SECRET)
  
  if (deployConfig.isSaaS()) {
    req.tenantId = decoded.tenantId
    // 验证租户状态
  } else {
    req.tenantId = undefined
  }
  
  next()
}
```

### 数据隔离
```typescript
// BaseRepository
async find(options) {
  const where = options?.where || {}
  
  if (deployConfig.isSaaS() && this.req.tenantId) {
    where['tenant_id'] = this.req.tenantId
  }
  
  return this.repository.find({ ...options, where })
}
```

### 模式判断
```typescript
// 部署模式配置
export const deployConfig = {
  mode: process.env.DEPLOY_MODE || 'private',
  isPrivate: () => deployConfig.mode === 'private',
  isSaaS: () => deployConfig.mode === 'saas'
}
```
