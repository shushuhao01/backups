# 私有客户模块 - 快速参考

## 📋 模块概述

私有客户模块用于管理私有部署客户的信息和授权，是Admin后台的核心功能之一。

**完成时间**: 2026-03-04  
**完成度**: 95%  
**状态**: ✅ 已完成开发，待测试

---

## 🎯 核心功能

### 1. 客户管理
- 创建私有客户（同时生成授权）
- 编辑客户信息
- 删除客户（有有效授权时拒绝）
- 搜索和筛选客户

### 2. 授权管理
- 生成新授权
- 续期授权
- 吊销授权
- 重新激活授权
- 查看授权历史
- 查看操作日志

## ⚠️ 注意事项

### 1. 授权码管理
- 授权码是私有部署系统激活的唯一凭证
- 创建客户时会自动生成授权码
- 授权码应妥善保管，避免泄露

### 2. 客户删除
- 只能删除没有有效授权的客户
- 删除客户前需要先吊销所有授权
- 删除操作不可恢复

### 3. 授权状态
- `pending`: 待激活（已生成但未激活）
- `active`: 有效（已激活且未过期）
- `expired`: 已过期（超过到期时间）
- `revoked`: 已吊销（手动停用）

### 4. 授权类型
- `trial`: 试用版（通常30天）
- `annual`: 年度版（1年有效期）
- `perpetual`: 永久版（无到期时间）
