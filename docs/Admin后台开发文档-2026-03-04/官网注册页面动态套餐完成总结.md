# 官网注册页面动态套餐完成总结

## 任务概述

将官网注册页面（Register.vue）的硬编码套餐数据改为从API动态获取，确保数据库、Admin后台、官网三个位置的套餐数据完全一致。

## 问题分析

### 原有问题

1. **硬编码的月付价格**
```typescript
const monthlyPrices: Record<string, number> = {
  basic: 99,
  pro: 299,
  enterprise: 599
}
```

2. **硬编码的私有部署价格**
```typescript
const privatePrices: Record<string, string> = {
  'private-standard': '9,800',
  'private-pro': '19,800',
  'private-enterprise': '39,800'
}
```

3. **硬编码的套餐卡片HTML**
   - SaaS套餐卡片写死了4个套餐
   - 私有部署套餐卡片写死了3个套餐
   - 套餐名称、价格、描述都是硬编码

4. **套餐代码映射不一致**
   - 前端使用 `basic`, `pro`, `enterprise`
   - 后端使用 `SAAS_BASIC`, `SAAS_PRO`, `SAAS_ENTERPRISE`
   - 需要额外的映射对象转换

## 实施方案

### 1. 修改数据获取方式

**修改前:**
```typescript
// 硬编码价格
const monthlyPrices: Record<string, number> = {
  basic: 99,
  pro: 299,
  enterprise: 599
}
```

**修改后:**
```typescript
// 从API动态获取
import { getPackages, type Package } from '@/api/packages'

const packages = ref<Package[]>([])
const loadingPackages = ref(true)

// 根据类型过滤套餐
const saasPackages = computed(() => packages.value.filter(p => p.type === 'saas'))
const privatePackages = computed(() => packages.value.filter(p => p.type === 'private'))

// 加载套餐数据
const loadPackages = async () => {
  loadingPackages.value = true
  try {
    packages.value = await getPackages()
  } catch (error) {
    console.error('加载套餐失败:', error)
  } finally {
    loadingPackages.value = false
  }
}

onMounted(async () => {
  await loadPackages()
})
```

### 2. 修改价格计算方法

**修改前:**
```typescript
const getMonthlyPrice = (plan: string) => {
  return monthlyPrices[plan] || 0
}

const getPlanPrice = (plan: string) => {
  if (plan === 'FREE_TRIAL') return '0'
  
  const privatePrices: Record<string, string> = {
    'private-standard': '9,800',
    'private-pro': '19,800',
    'private-enterprise': '39,800'
  }
  if (privatePrices[plan]) return privatePrices[plan]
  
  if (billingCycle.value === 'yearly') {
    return getYearlyPrice(plan).toLocaleString()
  }
  return getMonthlyPrice(plan).toString()
}
```

**修改后:**
```typescript
// 根据code查找套餐
const findPackageByCode = (code: string): Package | undefined => {
  return packages.value.find(p => p.code === code)
}

const getMonthlyPrice = (code: string) => {
  const pkg = findPackageByCode(code)
  return pkg?.price || 0
}

const getPlanPrice = (code: string) => {
  if (code === 'FREE_TRIAL') return '0'

  const pkg = findPackageByCode(code)
  if (!pkg) return '0'

  // 私有部署版直接返回价格
  if (pkg.type === 'private') {
    return pkg.price.toLocaleString()
  }

  // SaaS套餐根据计费周期返回
  if (billingCycle.value === 'yearly') {
    return getYearlyPrice(code).toLocaleString()
  }
  return pkg.price.toString()
}
```
