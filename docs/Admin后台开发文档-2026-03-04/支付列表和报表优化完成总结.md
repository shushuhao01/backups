## 🔧 技术实现

### 1. 部分退款实现

```typescript
const handleRefund = (row: any, type: 'full' | 'partial') => {
  if (type === 'partial') {
    // 部分退款：输入金额
    ElMessageBox.prompt('请输入退款金额', '部分退款', {
      inputPattern: /^\d+(\.\d{1,2})?$/,
      inputErrorMessage: '请输入正确的金额'
    }).then(async ({ value }) => {
      const amount = parseFloat(value)
      if (amount <= 0 || amount > row.amount) {
        ElMessage.error('退款金额不正确')
        return
      }
      // 调用退款API
      await request.post(`/payment/orders/${row.id}/refund`, { 
        amount,
        type: 'partial',
        reason: '部分退款'
      })
    })
  } else {
    // 全额退款：输入原因
    ElMessageBox.prompt('请输入退款原因', '全额退款')
      .then(async ({ value }) => {
        await request.post(`/payment/orders/${row.id}/refund`, { 
          type: 'full',
          reason: value 
        })
      })
  }
}
```

### 2. CSV导出实现

```typescript
// 前端
const handleExport = async () => {
  const params = new URLSearchParams({
    startDate: filters.dateRange[0],
    endDate: filters.dateRange[1],
    payType: filters.payType,
    customerType: filters.customerType
  })
  
  const token = localStorage.getItem('admin_token')
  const response = await fetch(`/api/v1/admin/payment/export?${params}`, {
    headers: { 'Authorization': `Bearer ${token}` }
  })
  
  const blob = await response.blob()
  const url = window.URL.createObjectURL(blob)
  const a = document.createElement('a')
  a.href = url
  a.download = `payment_orders_${Date.now()}.csv`
  a.click()
}
```

```typescript
// 后端
async exportData(req: Request, res: Response) {
  const orders = await orderRepo.find(/* 查询条件 */)
  
  // 构建CSV数据
  const csvData = [
    ['订单号', '租户名称', '套餐名称', ...], // 表头
    ...orders.map(order => [
      order.orderNo,
      order.tenantName,
      order.packageName,
      ...
    ])
  ]
  
  // 转换为CSV字符串
  const csvContent = csvData.map(row => 
    row.map(cell => {
      // 处理特殊字符
      if (cell.includes(',') || cell.includes('"')) {
        return `"${cell.replace(/"/g, '""')}"`
      }
      return cell
    }).join(',')
  ).join('\n')
  
  // 添加BOM头支持Excel中文
  res.send('\uFEFF' + csvContent)
}
```

## 📝 使用说明

### 1. 支付列表操作

**筛选订单**：
1. 输入订单号（模糊搜索）
2. 选择支付方式（微信/支付宝/对公转账）
3. 选择客户类型（私有客户/租户客户）
4. 选择订单状态
5. 选择时间范围
6. 点击"搜索"按钮

**退款操作**：
- 全额退款：点击"全额退款"按钮 → 输入退款原因 → 确认
- 部分退款：点击"部分退款"按钮 → 输入退款金额 → 确认

**导出数据**：
1. 设置筛选条件（可选）
2. 点击"导出"按钮
3. 下载CSV文件
4. 使用Excel打开（自动识别中文）

### 2. 支付报表操作

**查看报表**：
1. 选择时间范围或使用快捷按钮
2. 选择支付方式（可选）
3. 选择客户类型（可选）
4. 点击"查询"按钮

**切换时间维度**：
- 收入趋势图右上角选择：按日/按月/按年

**导出报表**：
1. 设置筛选条件
2. 点击"导出报表"按钮
3. 下载CSV文件
