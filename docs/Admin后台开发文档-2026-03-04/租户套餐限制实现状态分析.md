## 风险评估

### 高风险（立即修复）
1. **用户数无限制** - 租户可以无限创建用户
2. **存储空间无限制** - 租户可以无限上传文件
3. **过期不检查** - 过期租户仍可正常使用

### 中风险（尽快修复）
4. **功能权限无控制** - 所有租户都能访问所有功能
5. **存储空间不统计** - 无法准确计费和限制

### 低风险（后续优化）
6. **资源使用监控** - 缺少资源使用报表和告警

#### 步骤3: 添加过期检查
```typescript
// backend/src/middleware/tenantAuth.ts
export const tenantAuth = async (req: TenantRequest, res: Response, next: NextFunction) => {
  // ... 现有的认证逻辑 ...
  
  // 添加过期检查
  if (tenant.expire_date) {
    const now = new Date()
    const expireDate = new Date(tenant.expire_date)
    
    if (now > expireDate) {
      return res.status(403).json({
        success: false,
        message: '您的套餐已过期，请续费后继续使用',
        code: 'TENANT_EXPIRED',
        data: {
          expireDate: tenant.expire_date,
          daysExpired: Math.floor((now.getTime() - expireDate.getTime()) / (1000 * 60 * 60 * 24))
        }
      })
    }
  }
  
  next()
}
```

  REPORT: 'report',     // 报表分析

export const checkFeature = (feature: string) => {
  return async (req: TenantRequest, res: Response, next: NextFunction) => {
    const tenant = await getTenant(req.tenantId)
    const features = tenant.features ? JSON.parse(tenant.features) : []

}
