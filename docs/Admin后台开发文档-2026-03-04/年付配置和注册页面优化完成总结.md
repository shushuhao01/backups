## 一、年付配置功能

### 1.1 数据库字段添加

**文件:** `backend/database-migrations/add-yearly-discount-config.sql`

添加了3个新字段到`packages`表:

```sql
ALTER TABLE `packages` 
ADD COLUMN `yearly_discount_rate` DECIMAL(5, 2) DEFAULT 0.00 COMMENT '年付折扣率（0-100，例如20表示8折）',
ADD COLUMN `yearly_bonus_months` INT DEFAULT 0 COMMENT '年付赠送月数',
ADD COLUMN `yearly_price` DECIMAL(10, 2) DEFAULT NULL COMMENT '年付价格（如果为NULL则自动计算）';
```

**字段说明:**
- `yearly_discount_rate`: 年付折扣率（暂未使用，预留）
- `yearly_bonus_months`: 年付赠送月数（例如：2表示送2个月）
- `yearly_price`: 年付价格（如果为NULL，则自动计算为：月付价格 × (12 - 赠送月数)）

**默认配置:**
- SaaS月付套餐: 赠送2个月，年付价格 = 月付价格 × 10
- 免费试用套餐: 不需要年付配置
- 私有部署版: 一次性付费，不需要年付配置

### 1.2 实体类更新

**文件:** `backend/src/entities/Package.ts`

```typescript
@Column({ type: 'decimal', precision: 5, scale: 2, default: 0, comment: '年付折扣率' })
yearly_discount_rate!: number;

@Column({ type: 'int', default: 0, comment: '年付赠送月数' })
yearly_bonus_months!: number;

@Column({ type: 'decimal', precision: 10, scale: 2, nullable: true, comment: '年付价格' })
```

### 1.4 官网注册页面使用年付配置

**文件:** `website/src/views/Register.vue`

```typescript
// 年付价格（使用API返回的配置）
const getYearlyPrice = (code: string) => {
  const pkg = findPackageByCode(code)
  if (!pkg) return 0
  
  // 如果有配置年付价格,直接使用
  if (pkg.yearly_price) {
    return pkg.yearly_price
  }
  
  // 否则根据赠送月数计算: 月付价格 × (12 - 赠送月数)
  const bonusMonths = pkg.yearly_bonus_months || 2
  return pkg.price * (12 - bonusMonths)
}

// 年付节省金额
const getSaveAmount = (code: string) => {
  const pkg = findPackageByCode(code)
  if (!pkg) return 0
  
  const bonusMonths = pkg.yearly_bonus_months || 2
  return pkg.price * bonusMonths
}
```

## 二、注册页面优化

### 2.1 移除硬编码的"7天免费试用"

**问题:** 注册页面硬编码了一个"7天免费试用"卡片,与API返回的免费试用套餐重复

**修改前:**
```vue
<!-- 硬编码的免费试用卡片 -->
<div class="plan-card free-trial" @click="selectedPlan = 'FREE_TRIAL'">
  <h4>7天免费试用</h4>
  <p class="plan-price free">¥0</p>
</div>

<!-- API动态渲染的套餐 -->
<div v-for="pkg in saasPackages" :key="pkg.id" class="plan-card">
  ...
</div>
```

**修改后:**
```vue
<!-- 统一动态渲染所有套餐（包括免费试用） -->
<div
  v-for="pkg in saasPackages"
  :key="pkg.id"
  class="plan-card"
  :class="{ 
    selected: selectedPlan === pkg.code,
    'free-trial': pkg.is_trial || pkg.price == 0
  }"
  @click="selectedPlan = pkg.code"
>
  <div v-if="pkg.is_trial || pkg.price == 0" class="plan-badge free">免费</div>
  <div v-else-if="pkg.is_recommended" class="plan-badge">推荐</div>
  <h4>{{ pkg.name }}</h4>
  <p class="plan-price" :class="{ free: pkg.price == 0 }">
    ¥{{ pkg.price }}<span v-if="pkg.price > 0">/月</span>
  </p>
  <p class="plan-desc">
    <template v-if="pkg.price == 0">
      {{ pkg.duration_days }}天试用 · 体验全部功能
    </template>
    <template v-else>
      {{ pkg.max_users }}用户 · {{ pkg.max_storage_gb }}GB存储
    </template>
  </p>
</div>
```

## 七、注意事项

### 7.1 年付配置规则

1. **仅SaaS月付套餐支持年付配置**
   - 私有部署版是一次性付费,不需要年付
   - 免费试用套餐不需要年付

2. **赠送月数范围**
   - 最小: 0个月（无优惠）
   - 最大: 12个月（相当于免费）
   - 推荐: 2个月（相当于8.3折）

3. **年付价格计算**
   - 自动模式: 月付价格 × (12 - 赠送月数)
   - 手动模式: 使用自定义价格
   - 显示: 年付相当于多少/月,节省多少

### 7.2 免费套餐规则

1. **识别条件**
   - `is_trial = 1` 或 `price = 0`
   - 两个条件满足其一即可

2. **显示规则**
   - 显示"免费"标签
   - 显示试用天数
   - 不显示年付选项

3. **注册流程**
   - 直接跳转成功页
   - 不进入支付流程

### 7.3 数据同步

1. **Admin后台修改后**
   - 官网立即生效（API实时获取）
   - 无需重启服务
   - 无需清除缓存

2. **浏览器缓存**
