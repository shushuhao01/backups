### ✅ 1. 创建支付订单API

**文件**: `backend/src/controllers/public/PaymentController.ts`

**功能**:
- 创建支付订单
- 验证支付配置是否启用
- 调用微信/支付宝API生成二维码
- 返回二维码URL和支付链接
- 错误处理和日志记录

**API**: `POST /api/v1/public/payment/create`

**请求参数**:
```json
{
  "packageId": "basic",
  "packageName": "基础版",
  "amount": 99.00,
  "payType": "wechat",
  "billingCycle": "monthly",
  "tenantId": "xxx",
  "tenantName": "测试企业",
  "contactName": "张三",
  "contactPhone": "13800138000",
  "contactEmail": "test@example.com"
}
```

**响应**:
```json
{
  "code": 0,
  "message": "创建支付订单成功",
  "data": {
    "orderId": "xxx",
    "orderNo": "PAY1234567890",
    "qrCode": "weixin://wxpay/bizpayurl?pr=xxx",
    "payUrl": "weixin://wxpay/bizpayurl?pr=xxx",
    "expireTime": "2026-03-05 15:30:00"
  }
}
```

## 使用说明

### 1. 安装依赖

```bash
cd backend
npm install axios
```

或运行：
```bash
backend/install-payment-deps.bat
```

### 2. 配置支付方式

在Admin后台 → 支付管理 → 支付配置：

#### 微信支付配置
- AppID: 微信开放平台应用ID
- 商户号: 微信支付商户号
- API密钥V3: 微信支付API密钥
- 证书序列号: 商户证书序列号
- 回调地址: `https://your-domain.com/api/v1/admin/payment/notify/wechat`

#### 支付宝配置
- AppID: 支付宝开放平台应用ID
- 应用私钥: RSA2私钥
- 支付宝公钥: 支付宝RSA2公钥
- 签名类型: RSA2
- 回调地址: `https://your-domain.com/api/v1/admin/payment/notify/alipay`

### 3. 测试支付流程

```bash
# 启动后端服务
cd backend
npm run dev

# 运行测试脚本
node test-payment-flow.js
```

### 4. 开发模式 vs 生产模式

#### 开发模式
- 未配置API密钥时自动启用
- 返回模拟二维码（MOCK_WECHAT_xxx / MOCK_ALIPAY_xxx）
- 不调用真实支付API
- 方便本地开发测试

#### 生产模式
- 配置完整的API密钥后启用
- 调用真实支付API
- 生成真实支付二维码
- 需要公网可访问的回调地址

### 5. 回调地址配置

**重要**: 回调地址必须是公网可访问的HTTPS地址

#### 开发环境
使用内网穿透工具（如ngrok）：
```bash
ngrok http 3000
# 获得: https://xxx.ngrok.io

# 回调地址配置为:
# https://xxx.ngrok.io/api/v1/admin/payment/notify/wechat
# https://xxx.ngrok.io/api/v1/admin/payment/notify/alipay
```

#### 生产环境
```
https://your-domain.com/api/v1/admin/payment/notify/wechat
https://your-domain.com/api/v1/admin/payment/notify/alipay
```

## 注意事项

### 1. 安全性
- 所有支付配置加密存储
- 生产环境必须验证签名
- 回调接口做好幂等性处理
- 记录所有支付日志

### 2. 错误处理
- API调用失败返回友好提示
- 支付失败自动删除订单
- 回调失败记录错误日志
- 重复回调不重复处理

### 3. 性能优化
- 支付API调用设置超时（10秒）
- 订单30分钟自动过期
- 使用连接池优化数据库查询
