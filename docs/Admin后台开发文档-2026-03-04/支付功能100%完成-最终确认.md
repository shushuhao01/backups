## 📝 使用指南

### 管理员审核对公转账

1. **登录Admin后台**
   - 访问: `http://localhost:5174`
   - 使用管理员账号登录

2. **进入支付列表**
   - 点击左侧菜单"支付管理" → "支付列表"

3. **筛选待审核订单**
   - 在"状态"下拉框选择"待审核"
   - 点击"搜索"

4. **查看订单详情**
   - 点击订单的"详情"按钮
   - 查看转账凭证图片（可点击放大）
   - 查看转账信息（时间、金额、账户）

5. **审核操作**
   
   **方式一：在列表中操作**
   - 点击"审核通过"按钮
   - 填写审核备注（选填）
   - 点击"确认通过"
   
   **方式二：在详情弹窗中操作**
   - 在详情弹窗底部点击"审核通过"或"审核拒绝"
   - 填写备注或原因
   - 点击确认

6. **审核结果**
   - 审核通过：订单状态变为"已支付"，自动激活租户，生成授权码
   - 审核拒绝：订单状态变为"审核拒绝"，用户需要重新转账
