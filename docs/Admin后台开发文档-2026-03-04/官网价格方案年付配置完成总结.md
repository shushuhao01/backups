# 官网价格方案年付配置完成总结

## 问题描述

官网首页的"价格方案"页面（`website/src/views/Pricing.vue`）使用了硬编码的年付计算逻辑：
- 年付价格固定使用20%折扣（8折）
- 没有使用数据库配置的 `yearly_bonus_months` 和 `yearly_price` 字段
- 没有显示赠送月数信息
- 节省金额计算不准确

## 修复内容

### 1. 更新 API 接口文件 (`website/src/api/packages.ts`)

**修改前（硬编码逻辑）:**
```typescript
// 计算年付价格（8折）
export function getYearlyPrice(monthlyPrice: number): number {
  return Math.round(monthlyPrice * 0.8)
}

// 计算年付总价
export function getYearlyTotal(monthlyPrice: number): number {
  return getYearlyPrice(monthlyPrice) * 12
}

// 计算年付节省金额
export function getYearlySaving(monthlyPrice: number): number {
  return monthlyPrice * 12 - getYearlyTotal(monthlyPrice)
}
```

**修改后（使用数据库配置）:**
```typescript
// 计算年付价格（使用套餐配置）
export function getYearlyPrice(pkg: Package): number {
  // 如果有配置年付价格，直接使用
  if (pkg.yearly_price) {
    return pkg.yearly_price
  }
  // 否则根据赠送月数计算: 月付价格 × (12 - 赠送月数)
  const bonusMonths = pkg.yearly_bonus_months || 0
  return pkg.price * (12 - bonusMonths)
}

// 计算年付总价
export function getYearlyTotal(pkg: Package): number {
  return getYearlyPrice(pkg)
}

// 计算年付节省金额（动态计算）
export function getYearlySaving(pkg: Package): number {
  // 节省金额 = 正常年付总价 - 实际年付价格
  return pkg.price * 12 - getYearlyPrice(pkg)
}
```

## 计算逻辑说明

### 年付价格计算
```javascript
// 优先使用数据库配置的年付价格
if (pkg.yearly_price) {
  return pkg.yearly_price
}
// 否则根据赠送月数自动计算
return pkg.price * (12 - pkg.yearly_bonus_months)
```

### 节省金额计算
```javascript
// 节省金额 = 正常年付总价 - 实际年付价格
return pkg.price * 12 - getYearlyPrice(pkg)
```

### 折算月价计算
```javascript
// 年付价格 ÷ 12
return Math.round(getYearlyPrice(pkg) / 12)
```
