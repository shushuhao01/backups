## 📋 API接口速查

### 支付管理（12个接口）

```bash
# 配置管理
GET    /api/admin/payment/config
POST   /api/admin/payment/config/wechat
POST   /api/admin/payment/config/alipay

# 订单管理
POST   /api/admin/payment/orders              # 创建订单
GET    /api/admin/payment/orders              # 订单列表
GET    /api/admin/payment/orders/:id          # 订单详情
PUT    /api/admin/payment/orders/:id          # 更新订单
POST   /api/admin/payment/orders/:id/close    # 关闭订单
POST   /api/admin/payment/orders/:id/refund   # 退款
POST   /api/admin/payment/orders/batch-refund # 批量退款

# 统计日志
GET    /api/admin/payment/stats               # 支付统计
GET    /api/admin/payment/logs                # 支付日志
```

### 统计数据（5个接口）

```bash
GET    /api/admin/statistics/dashboard        # 仪表盘
GET    /api/admin/statistics/tenants          # 租户统计
GET    /api/admin/statistics/revenue          # 收入统计
GET    /api/admin/statistics/users            # 用户统计
GET    /api/admin/statistics/trend            # 趋势分析
```

## 📊 仪表盘数据结构

```typescript
{
  tenants: {
    total: number,        // 租户总数
    active: number,       // 活跃租户
    expired: number,      // 过期租户
    newThisMonth: number  // 本月新增
  },
  licenses: {
    total: number,        // 授权总数
    active: number,       // 活跃授权
    expired: number,      // 过期授权
    trial: number,        // 试用授权
    expiringSoon: number  // 即将到期
  },
  revenue: {
    total: number,        // 总收入
    thisMonth: number,    // 本月收入
    today: number,        // 今日收入
    growthRate: string    // 增长率
  },
  orders: {
    total: number,        // 订单总数
    pending: number,      // 待支付
    paid: number          // 已支付
  },
  privateCustomers: {
    total: number,        // 私有客户总数
    active: number        // 活跃客户
  }
}
```
