## 📊 数据库表设计

### 1. licenses表（授权记录表）

**表名**: `licenses`  
**用途**: 存储授权记录，用于私有客户的授权管理  
**引擎**: InnoDB  
**字符集**: utf8mb4_unicode_ci

#### 字段设计

| 字段名 | 类型 | 约束 | 说明 |
|--------|------|------|------|
| id | VARCHAR(36) | PRIMARY KEY | 授权ID（UUID）|
| license_key | VARCHAR(255) | NOT NULL UNIQUE | 授权码（唯一）|
| customer_name | VARCHAR(100) | NOT NULL | 客户名称 |
| customer_contact | VARCHAR(100) | | 联系人 |
| customer_phone | VARCHAR(20) | | 联系电话 |
| customer_email | VARCHAR(100) | | 邮箱 |
| license_type | ENUM | DEFAULT 'trial' | 授权类型 |
| max_users | INT | DEFAULT 10 | 最大用户数 |
| max_storage_gb | INT | DEFAULT 5 | 最大存储空间(GB) |
| features | JSON | | 开通的功能模块 |
| machine_id | VARCHAR(255) | | 绑定的机器码 |
| status | ENUM | DEFAULT 'pending' | 状态 |
| activated_at | DATETIME | | 激活时间 |
| expires_at | DATETIME | | 到期时间 |
| notes | TEXT | | 备注 |
| created_by | VARCHAR(36) | | 创建人 |
| created_at | DATETIME | DEFAULT CURRENT_TIMESTAMP | 创建时间 |
| updated_at | DATETIME | ON UPDATE CURRENT_TIMESTAMP | 更新时间 |

#### 枚举值说明

**license_type（授权类型）**:
- `trial`: 试用版
- `perpetual`: 永久授权
- `annual`: 年度授权
- `monthly`: 月度授权

**status（状态）**:
- `pending`: 待激活
- `active`: 已激活
- `expired`: 已过期
- `revoked`: 已撤销

#### 索引设计

| 索引名 | 字段 | 类型 | 说明 |
|--------|------|------|------|
| PRIMARY | id | 主键 | 主键索引 |
| idx_license_key | license_key | 普通索引 | 授权码查询 |
| idx_status | status | 普通索引 | 状态筛选 |

---

### 2. license_logs表（授权验证日志表）

**表名**: `license_logs`  
**用途**: 记录授权验证、激活、续期等操作日志  
**引擎**: InnoDB  
**字符集**: utf8mb4_unicode_ci

#### 字段设计

| 字段名 | 类型 | 约束 | 说明 |
|--------|------|------|------|
| id | VARCHAR(36) | PRIMARY KEY | 日志ID（UUID）|
| license_id | VARCHAR(36) | | 授权ID |
| license_key | VARCHAR(255) | | 授权码 |
| action | ENUM | | 操作类型 |
| machine_id | VARCHAR(255) | | 机器码 |
| ip_address | VARCHAR(50) | | IP地址 |
| user_agent | TEXT | | User-Agent |
| result | ENUM | | 结果 |
| message | TEXT | | 消息 |
| created_at | DATETIME | DEFAULT CURRENT_TIMESTAMP | 创建时间 |

#### 枚举值说明

**action（操作类型）**:
- `verify`: 验证授权
- `activate`: 激活授权
- `renew`: 续期授权
- `revoke`: 撤销授权
- `expire`: 授权过期

**result（结果）**:
- `success`: 成功
- `failed`: 失败

#### 索引设计

| 索引名 | 字段 | 类型 | 说明 |
|--------|------|------|------|
| PRIMARY | id | 主键 | 主键索引 |
| idx_license_id | license_id | 普通索引 | 授权ID查询 |
| idx_created_at | created_at | 普通索引 | 时间范围查询 |
