- 格式：XXXX-XXXX-XXXX-XXXX-XXXX
- 使用crypto模块生成随机密钥

### 2. 授权管理
- 支持4种授权类型（trial, perpetual, annual, monthly）
- 支持4种状态（pending, active, expired, revoked）
- 支持资源限制（用户数、存储空间）
- 支持功能模块配置（JSON格式）

### 3. 授权验证
- 验证授权码存在性
- 验证授权状态
- 验证到期时间
- 验证机器码绑定
- 自动记录验证日志

### 4. 授权日志
- 记录所有操作（verify, activate, renew, revoke, expire）
- 记录操作结果（success, failed）
- 记录环境信息（machineId, ipAddress, userAgent）
- 支持分页查询

### 5. 统计功能
- 总授权数
- 各状态授权数（active, pending, expired, revoked）

## 🚀 API使用示例

### 创建授权
```bash
POST /api/admin/licenses
Content-Type: application/json
Authorization: Bearer <token>

{
  "customerName": "测试客户",
  "customerContact": "张三",
  "customerPhone": "13800138000",
  "customerEmail": "test@example.com",
  "licenseType": "trial",
  "maxUsers": 20,
  "maxStorageGb": 10,
  "features": ["订单管理", "客户管理"],
  "expiresAt": "2026-04-04",
  "notes": "测试授权"
}
```

### 获取授权列表
```bash
GET /api/admin/licenses?page=1&pageSize=20&keyword=测试&status=active
Authorization: Bearer <token>
```

### 激活授权
```bash
POST /api/admin/licenses/:id/activate
Content-Type: application/json
Authorization: Bearer <token>

{
  "machineId": "MACHINE-001"
}
```

### 验证授权
```bash
POST /api/admin/licenses/verify
Content-Type: application/json

{
  "licenseKey": "XXXX-XXXX-XXXX-XXXX-XXXX",
  "machineId": "MACHINE-001"
}
```
