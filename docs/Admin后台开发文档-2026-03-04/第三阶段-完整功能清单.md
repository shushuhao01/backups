## 💡 使用示例

### 创建支付订单
```javascript
const response = await axios.post('/api/admin/payment/orders', {
  customerType: 'tenant',
  tenantName: '测试租户',
  packageName: '专业版',
  amount: 999.00,
  payType: 'wechat',
  contactName: '张三',
  contactPhone: '13800138000'
}, {
  headers: { 'Authorization': `Bearer ${token}` }
});
```

### 获取仪表盘统计
```javascript
const response = await axios.get('/api/admin/statistics/dashboard', {
  headers: { 'Authorization': `Bearer ${token}` }
});

console.log(response.data.data.revenue.total); // 总收入
console.log(response.data.data.tenants.active); // 活跃租户
```

### 导出支付数据
```javascript
const response = await axios.get('/api/admin/payment/export', {
  params: {
    status: 'paid',
    startDate: '2026-01-01',
    endDate: '2026-12-31'
  },
  headers: { 'Authorization': `Bearer ${token}` },
  responseType: 'blob'
});

// 下载CSV文件
const url = window.URL.createObjectURL(new Blob([response.data]));
const link = document.createElement('a');
link.href = url;
link.setAttribute('download', 'payment_orders.csv');
document.body.appendChild(link);
link.click();
```
