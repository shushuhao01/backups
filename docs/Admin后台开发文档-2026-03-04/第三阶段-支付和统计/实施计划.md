## ⚠️ 注意事项

### 数据库兼容性
- 所有SQL必须兼容phpMyAdmin/宝塔面板
- 避免使用动态SQL语法
- 使用简单的CREATE TABLE和ALTER TABLE

### 安全性
- 支付配置敏感信息加密存储
- API密钥脱敏显示
- 支付回调验证签名
- 防止SQL注入

### 性能优化
- 统计数据使用缓存
- 复杂查询添加索引
- 分页查询优化
- 避免N+1查询

### 错误处理
- 完善的错误提示
- 支付失败重试机制
- 日志记录完整
- 异常情况处理
