## 📊 API接口清单

### 支付管理API（已有基础，已完善）

#### 订单管理
```
POST   /api/admin/payment/orders              - 创建支付订单 ✅
GET    /api/admin/payment/orders              - 获取订单列表 ✅
GET    /api/admin/payment/orders/:id          - 获取订单详情 ✅
PUT    /api/admin/payment/orders/:id          - 更新订单 ✅
POST   /api/admin/payment/orders/:id/close    - 关闭订单 ✅
POST   /api/admin/payment/orders/:id/refund   - 退款 ✅
POST   /api/admin/payment/orders/batch-refund - 批量退款 ✅
```

#### 配置管理
```
GET    /api/admin/payment/config              - 获取支付配置 ✅
POST   /api/admin/payment/config/wechat       - 保存微信配置 ✅
POST   /api/admin/payment/config/alipay       - 保存支付宝配置 ✅
```

#### 统计和日志
```
GET    /api/admin/payment/stats               - 获取支付统计 ✅
GET    /api/admin/payment/logs                - 获取支付日志 ✅
```

### 统计数据API（新建）

```
GET    /api/admin/statistics/dashboard        - 仪表盘统计 ✅
```

## 🎯 统计数据说明

### 仪表盘统计数据结构
```typescript
{
  tenants: {
    total: number,           // 租户总数
    active: number,          // 活跃租户
    expired: number,         // 过期租户
    newThisMonth: number     // 本月新增
  },
  licenses: {
    total: number,           // 授权总数
    active: number,          // 活跃授权
    expired: number,         // 过期授权
    trial: number,           // 试用授权
    expiringSoon: number     // 即将到期
  },
  revenue: {
    total: number,           // 总收入
    thisMonth: number,       // 本月收入
    today: number,           // 今日收入
    growthRate: string       // 增长率
  },
  orders: {
    total: number,           // 订单总数
    pending: number,         // 待支付
    paid: number             // 已支付
  },
  privateCustomers: {
    total: number,           // 私有客户总数
    active: number           // 活跃客户
  }
}
```
