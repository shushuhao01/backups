## 🎯 使用指南

### 快速开始

#### 1. 安装依赖
```bash
cd backend
npm install axios
```

#### 2. 启动服务
```bash
npm run dev
```

#### 3. 测试支付流程
```bash
node test-payment-flow.js
```

### 配置支付方式

#### 微信支付
1. 登录Admin后台
2. 进入"支付管理" → "支付配置"
3. 选择"微信支付"标签
4. 填写配置信息
5. 点击"保存配置"

#### 支付宝
1. 登录Admin后台
2. 进入"支付管理" → "支付配置"
3. 选择"支付宝"标签
4. 填写配置信息
5. 点击"保存配置"

### 测试支付

#### 开发环境
- 无需配置真实支付参数
- 自动返回模拟二维码
- 可以测试完整流程

#### 生产环境
- 配置真实支付参数
- 配置公网回调地址
- 使用真实支付
