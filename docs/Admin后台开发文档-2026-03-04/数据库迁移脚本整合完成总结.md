## 📊 数据库表结构总览

### Admin后台核心表 (2个)

| 表名 | 说明 | 主要字段 |
|------|------|---------|
| admin_users | Admin用户 | username, password, role, status |
| admin_operation_logs | Admin操作日志 | admin_user_id, action, module |

### 租户管理表 (3个)

| 表名 | 说明 | 主要字段 |
|------|------|---------|
| tenants | 租户信息 | name, code, package_id, expire_date |
| tenant_settings | 租户设置 | tenant_id, setting_key, setting_value |
| tenant_logs | 租户日志 | tenant_id, action, description |

### 套餐管理表 (1个)

| 表名 | 说明 | 主要字段 |
|------|------|---------|
| packages | 套餐配置 | name, code, price, max_users, yearly_price |

### 授权管理表 (2个)

| 表名 | 说明 | 主要字段 |
|------|------|---------|
| licenses | 授权记录 | license_key, customer_name, status |
| license_logs | 授权日志 | license_id, action, result |

### 私有客户表 (1个)

| 表名 | 说明 | 主要字段 |
|------|------|---------|
| private_customers | 私有客户 | customer_name, contact_person, industry |

### 版本管理表 (2个)

| 表名 | 说明 | 主要字段 |
|------|------|---------|
| versions | 版本信息 | version, release_type, download_url |
| changelogs | 更新日志 | version_id, type, content |

### 字段补充 (2个表)

**payment_orders表:**
- `billing_cycle` VARCHAR(20) - 计费周期
- `bonus_months` INT - 赠送月数

**packages表:**
- `yearly_discount_rate` DECIMAL(5,2) - 年付折扣率
- `yearly_bonus_months` INT - 年付赠送月数
- `yearly_price` DECIMAL(10,2) - 年付价格

### 步骤1: 备份数据库
```bash
# 在宝塔面板中备份数据库
# 或使用命令行
mysqldump -u username -p database_name > backup_$(date +%Y%m%d_%H%M%S).sql
```

### 步骤2: 执行迁移脚本
```sql
-- 使用宝塔面板SQL执行功能
-- 或使用命令行
mysql -u username -p database_name < backend/database-migrations/production-admin-simple.sql
```
