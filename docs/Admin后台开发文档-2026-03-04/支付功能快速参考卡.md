# 支付功能快速参考卡

## 🎯 核心API

### 创建支付订单
```bash
POST /api/v1/public/payment/create
```
```json
{
  "packageId": "basic",
  "packageName": "基础版",
  "amount": 99.00,
  "payType": "wechat",
  "billingCycle": "monthly",
  "tenantId": "xxx",
  "tenantName": "测试企业",
  "contactName": "张三",
  "contactPhone": "13800138000",
  "contactEmail": "test@example.com"
}
```

### 查询支付状态
```bash
GET /api/v1/public/payment/query/:orderNo
```

### 取消订单
```bash
POST /api/v1/public/payment/cancel/:orderNo
```

## 🔧 配置步骤

### 1. 微信支付
Admin后台 → 支付管理 → 支付配置 → 微信支付

- AppID: 微信开放平台应用ID
- 商户号: 微信支付商户号
- API密钥V3: 32位字符串
- 证书序列号: 从证书获取
- 回调地址: `https://your-domain.com/api/v1/admin/payment/notify/wechat`

### 2. 支付宝
Admin后台 → 支付管理 → 支付配置 → 支付宝

- AppID: 支付宝开放平台应用ID
- 应用私钥: RSA2私钥
- 支付宝公钥: 支付宝RSA2公钥
- 签名类型: RSA2
- 回调地址: `https://your-domain.com/api/v1/admin/payment/notify/alipay`

## 📊 支付流程

```
官网注册 → 创建订单 → 生成二维码 → 用户扫码 → 支付成功
    ↓
回调处理 → 激活租户 → 生成授权码 → 查询状态 → 显示授权码
```

## 🔐 回调地址

### 微信支付
```
POST /api/v1/admin/payment/notify/wechat
```

### 支付宝
```
POST /api/v1/admin/payment/notify/alipay
```

**重要**: 必须是公网可访问的HTTPS地址

---

## 💡 开发模式 vs 生产模式

### 开发模式
- 未配置API密钥时自动启用
- 返回模拟二维码
- 不调用真实API
- 适合本地测试

### 生产模式
- 配置完整API密钥后启用
- 调用真实支付API
- 生成真实二维码
- 需要公网回调地址
