## 📊 数据库表结构

### modules表
| 字段 | 类型 | 说明 |
|------|------|------|
| id | VARCHAR(36) | 主键 |
| name | VARCHAR(100) | 模块名称 |
| code | VARCHAR(50) | 模块代码（唯一） |
| description | TEXT | 模块描述 |
| icon | VARCHAR(100) | 模块图标 |
| version | VARCHAR(20) | 模块版本 |
| status | ENUM | 状态（enabled/disabled） |
| is_system | BOOLEAN | 是否系统模块 |
| sort_order | INT | 排序 |
| created_at | TIMESTAMP | 创建时间 |
| updated_at | TIMESTAMP | 更新时间 |

### module_configs表
| 字段 | 类型 | 说明 |
|------|------|------|
| id | VARCHAR(36) | 主键 |
| module_id | VARCHAR(36) | 模块ID（外键） |
| config_key | VARCHAR(100) | 配置键 |
| config_value | TEXT | 配置值 |
| config_type | ENUM | 配置类型（string/number/boolean/json） |
| description | VARCHAR(255) | 配置说明 |
| created_at | TIMESTAMP | 创建时间 |
| updated_at | TIMESTAMP | 更新时间 |
