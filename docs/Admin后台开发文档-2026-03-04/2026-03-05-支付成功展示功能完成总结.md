## 🔧 技术实现

### 后端API

**文件**: `backend/src/controllers/public/PaymentController.ts`

**查询支付状态接口**:
```typescript
GET /api/v1/public/payment/query/:orderNo

返回数据:
{
  "code": 0,
  "data": {
    "orderNo": "PAY1234567890",
    "status": "paid",
    "amount": 1980,
    "payType": "wechat",
    "paidAt": "2026-03-05 14:30:00",
    "tenantCode": "T20260305001",
    "licenseKey": "XXXXXX-XXXXXX-XXXXXX-XXXXXX",
    "expireAt": "2027-03-05"
  }
}
```

**关键逻辑**:
1. 查询订单状态
2. 如果已支付，查询租户编码
3. 查询授权码（从licenses表）
4. 查询到期时间（从tenants表的expire_date字段）
5. 返回完整数据

### 前端轮询

**文件**: `website/src/views/Register.vue`

**轮询逻辑**:
```typescript
const startPaymentCheck = () => {
  checkingPayment.value = true
  paymentCheckTimer = setInterval(async () => {
    if (!paymentOrder.value) return

    try {
      const res = await fetch(`${API_BASE}/public/payment/query/${paymentOrder.value.orderNo}`)
      const data = await res.json()

      if (data.code === 0 && data.data.status === 'paid') {
        // 停止轮询
        clearInterval(paymentCheckTimer)
        clearInterval(expireTimer)
        checkingPayment.value = false

        // 跳转成功页面，传递所有必要信息
        router.push({
          path: '/pay-success',
          query: {
            plan: selectedPlan.value,
            type: planType.value,
            tenantCode: data.data.tenantCode,
            licenseKey: data.data.licenseKey,
            expireAt: data.data.expireAt || ''
          }
        })
      }
    } catch (error) {
      console.error('查询支付状态失败:', error)
    }
  }, 3000) // 每3秒查询一次
}
```

**关键特性**:
- 每3秒自动查询一次
- 检测到"已支付"后立即停止
- 自动跳转到成功页面
- 传递所有必要参数
