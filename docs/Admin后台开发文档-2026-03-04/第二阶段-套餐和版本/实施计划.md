#### 2.1.1 数据库设计（0.5天）
**时间**: 3月4日上午  
**负责人**: 后端开发

   - updated_at: 更新时间

2. 添加索引
   - idx_code: 套餐代码索引

## 💡 技术方案

### 套餐管理技术方案

#### 数据模型
```typescript
interface Package {
  id: string;
  name: string;
  code: string;
  description: string;
  price: number;
  priceYearly: number;
  maxUsers: number;
  maxStorageGb: number;
  features: string[];
  isPopular: boolean;
  sortOrder: number;
  status: 'active' | 'inactive';
  createdAt: Date;
  updatedAt: Date;
}
```

#### API设计
- RESTful风格
- 支持分页和筛选
- 返回统一格式

### 版本管理技术方案

#### 文件上传方案
- 使用multer中间件
- 支持本地存储和OSS存储
- 文件大小限制：500MB
- 支持的文件类型：.zip, .tar.gz, .exe, .dmg

#### 版本发布流程
1. 上传版本文件
2. 填写版本信息和更新日志
3. 预览版本信息
4. 发布版本
5. 通知相关用户
