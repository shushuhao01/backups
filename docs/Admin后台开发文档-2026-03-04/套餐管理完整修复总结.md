### 2. ❌ Admin后台编辑套餐时，checkbox字段保存后再打开没有勾选
**状态**: ✅ 已修复

**问题描述**:
- 编辑套餐时，is_trial、is_recommended、is_visible字段勾选后保存
- 再次打开编辑对话框，这些checkbox没有勾选状态

**根本原因**:
- 数据库返回的是数字0/1
- 前端`handleEdit`方法没有将0/1转换为boolean
- 导致checkbox绑定的值不正确

**解决方案**:
```typescript
// admin/src/views/tenant-customers/Packages.vue
const handleEdit = (pkg: Package) => {
  editingId.value = pkg.id
  Object.assign(form, {
    ...pkg,
    status: (pkg.status === 'active' || pkg.status === true || pkg.status === 1) ? 'active' : 'disabled',
    features: pkg.features?.length ? [...pkg.features] : [''],
    // 确保boolean字段正确转换
    is_trial: Boolean(pkg.is_trial),
    is_recommended: Boolean(pkg.is_recommended),
    is_visible: Boolean(pkg.is_visible)
  })
  showDialog.value = true
}
```

### 3. ❌ 保存时boolean字段类型不匹配
**状态**: ✅ 已修复

**问题描述**:
- 前端发送Boolean(true/false)到后端
- 数据库需要tinyint(0/1)
- 可能导致保存失败或数据不一致

**解决方案**:
```typescript
// 提交时转换为数字
const data = {
  // ... 其他字段
  is_trial: form.is_trial ? 1 : 0,
  is_recommended: form.is_recommended ? 1 : 0,
  is_visible: form.is_visible ? 1 : 0,
  status: form.status === 'active' ? 1 : 0
}
```

## 📋 完整的数据流

### 创建套餐流程
```
1. Admin后台填写表单
   ↓
2. 前端转换数据类型
   - boolean → 0/1
   - string status → 0/1
   ↓
3. POST /api/v1/admin/packages
   ↓
4. 后端保存到数据库
   ↓
5. 官网API自动可获取（如果启用且可见）
```

### 编辑套餐流程
```
1. 点击编辑按钮
   ↓
2. GET /api/v1/admin/packages/:id
   ↓
3. 后端返回数据（0/1格式）
   ↓
4. 前端转换显示
   - 0/1 → boolean (checkbox)
   - 0/1 → 'active'/'disabled' (status)
   ↓
5. 用户修改
   ↓
6. 前端转换提交
   - boolean → 0/1
   - 'active'/'disabled' → 0/1
   ↓
7. PUT /api/v1/admin/packages/:id
   ↓
8. 保存成功，官网立即更新
```

### 2. 后端 - 公开套餐API

#### backend/src/routes/public/packages.ts
```typescript
// 获取公开套餐列表
router.get('/', async (req, res) => {
  const queryBuilder = packageRepo
    .createQueryBuilder('package')
    .where('package.status = :status', { status: 1 })      // 只返回启用的
    .andWhere('package.is_visible = :visible', { visible: 1 })  // 只返回可见的
    .orderBy('package.sort_order', 'ASC')
    .addOrderBy('package.id', 'ASC');

  const packages = await queryBuilder.getMany();
  res.json({ code: 0, message: 'success', data: packages });
});
```

### packages表关键字段

| 字段 | 类型 | 说明 | 前端类型 | 转换规则 |
|------|------|------|----------|----------|
| status | tinyint(1) | 启用状态 | string | 'active'→1, 'disabled'→0 |
| is_trial | tinyint(1) | 是否试用 | boolean | true→1, false→0 |
| is_recommended | tinyint(1) | 是否推荐 | boolean | true→1, false→0 |

### 官网显示条件
```sql
WHERE status = 1 AND is_visible = 1
ORDER BY sort_order ASC, id ASC
```
