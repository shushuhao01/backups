## 🔄 支付流程

```
选择套餐 → 填写信息 → 创建订单 → 显示二维码
    ↓
开始轮询（每3秒）
    ↓
用户扫码支付
    ↓
支付平台回调 → 更新订单状态
    ↓
轮询检测到"已支付"
    ↓
自动跳转成功页面
    ↓
展示完整信息
```

## ⚡ 轮询机制

```typescript
// 每3秒查询一次支付状态
paymentCheckTimer = setInterval(async () => {
  const res = await fetch(`/api/v1/public/payment/query/${orderNo}`)
  const data = await res.json()

  if (data.code === 0 && data.data.status === 'paid') {
    // 停止轮询
    clearInterval(paymentCheckTimer)
    
    // 跳转成功页面
    router.push({
      path: '/pay-success',
      query: {
        plan: selectedPlan,
        type: planType,
        tenantCode: data.data.tenantCode,
        licenseKey: data.data.licenseKey,
        expireAt: data.data.expireAt
      }
    })
  }
}, 3000)
```
