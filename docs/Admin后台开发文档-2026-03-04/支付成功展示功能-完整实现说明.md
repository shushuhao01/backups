### 2. 官网注册页面支付轮询

**页面**: `website/src/views/Register.vue`

**功能**:
- ✅ 创建支付订单后自动开始轮询
- ✅ 每3秒查询一次支付状态
- ✅ 支付成功后自动跳转到成功页面
- ✅ 传递租户编码和授权码到成功页面

**轮询逻辑**:
```typescript
// 开始轮询支付状态
const startPaymentCheck = () => {
  checkingPayment.value = true
  paymentCheckTimer = setInterval(async () => {
    if (!paymentOrder.value) return

    try {
      const res = await fetch(`${API_BASE}/public/payment/query/${paymentOrder.value.orderNo}`)
      const data = await res.json()

      if (data.code === 0 && data.data.status === 'paid') {
        clearInterval(paymentCheckTimer)
        clearInterval(expireTimer)
        checkingPayment.value = false

        // 支付成功，从API获取授权码，跳转成功页
        router.push({
          path: '/pay-success',
          query: {
            plan: selectedPlan.value,
            type: planType.value,
            tenantCode: data.data.tenantCode,
            licenseKey: data.data.licenseKey
          }
        })
      }
    } catch (error) {
      console.error('查询支付状态失败:', error)
    }
  }, 3000) // 每3秒查询一次
}
```

**订单过期倒计时**:
- ✅ 30分钟订单有效期
- ✅ 实时显示剩余时间
- ✅ 过期后自动停止轮询

## 🔄 完整支付流程

### 流程图

```
用户选择套餐
    ↓
填写注册信息
    ↓
创建支付订单
    ↓
显示支付二维码
    ↓
[开始轮询] ← 每3秒查询一次
    ↓
用户扫码支付
    ↓
支付平台回调 → 更新订单状态为"已支付"
    ↓
轮询检测到"已支付"
    ↓
停止轮询
    ↓
跳转支付成功页面
    ↓
展示完整租户信息
```

### 关键时间点

1. **订单创建**: 生成订单号和二维码
2. **开始轮询**: 立即开始，每3秒一次
3. **支付完成**: 支付平台回调更新订单状态
4. **检测成功**: 轮询检测到状态变化（最多延迟3秒）
5. **跳转页面**: 自动跳转到成功页面
6. **展示信息**: 显示所有必要信息

### 1. 权益到期时间显示

**当前状态**: 后端已计算到期时间，但未在成功页面显示

**需要补充**:
1. 在支付状态查询API中返回 `expireAt` 字段
2. 在成功页面显示到期时间

**实现方案**:

#### 后端修改
```typescript
// backend/src/controllers/public/PaymentController.ts
async queryOrder(req: Request, res: Response) {
  // ... 现有代码 ...
  
  // 查询租户信息获取到期时间
  let expireAt = null
  if (order.tenant_id) {
    const tenants = await AppDataSource.query(
      'SELECT expire_at FROM tenants WHERE id = ?',
      [order.tenant_id]
    )
    if (tenants.length > 0) {
      expireAt = tenants[0].expire_at
    }
  }

  res.json({
    code: 0,
    data: {
      orderNo: order.order_no,
      status: order.status,
      amount: order.amount,
      payType: order.pay_type,
      paidAt: order.paid_at,
      tenantCode: order.tenant_code,
      licenseKey: licenseKey,
      expireAt: expireAt  // 新增
    }
  })
}
```

#### 前端修改
```vue
<!-- website/src/views/PaySuccess.vue -->
<template>
  <!-- ... 现有代码 ... -->
  <div class="info-item" v-if="expireAt">
    <span class="info-label">权益到期时间</span>
    <span class="info-value">{{ formatDate(expireAt) }}</span>
  </div>
  <!-- ... 现有代码 ... -->
</template>

<script setup lang="ts">
// 从URL参数获取到期时间
const expireAt = computed(() => route.query.expireAt as string || '')

// 格式化日期
const formatDate = (dateStr: string) => {
  if (!dateStr) return ''
  const date = new Date(dateStr)
  return date.toLocaleDateString('zh-CN', {
    year: 'numeric',
    month: '2-digit',
    day: '2-digit'
  })
}
</script>
```
