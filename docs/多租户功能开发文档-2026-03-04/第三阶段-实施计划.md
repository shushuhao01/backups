### 2. 租户数据查询（Admin视角）

```typescript
// Admin可以查询所有租户的数据
// 不使用BaseRepository，直接使用原生Repository
const tenantRepo = AppDataSource.getRepository(Tenant);
const tenants = await tenantRepo.find({
  where: { status: 'active' }
  // 不添加tenant_id过滤
});
```

## 📝 注意事项

1. **Admin权限**：
   - Admin用户不受租户限制
   - 可以访问所有租户数据
   - 需要特殊的权限验证中间件

2. **数据安全**：
   - 租户数据必须隔离
   - Admin操作需要记录日志
   - 敏感信息需要脱敏

3. **性能考虑**：
   - 租户列表需要分页
   - 统计数据需要缓存
   - 避免N+1查询

4. **用户体验**：
   - 操作需要确认
   - 错误提示友好
   - 加载状态明确
