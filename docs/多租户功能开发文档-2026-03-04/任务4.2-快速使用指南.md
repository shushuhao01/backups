# 任务 4.2：租户数据导入 - 快速使用指南

## 🚀 快速开始

### 1. 测试导入功能
```bash
cd backend
node test-tenant-import.js
```

### 2. API 使用

#### 创建导入任务
```bash
curl -X POST \
  http://localhost:3000/api/v1/admin/tenants/TENANT_ID/import \
  -H "Authorization: Bearer YOUR_TOKEN" \
  -F "file=@export_data.json" \
  -F "conflictStrategy=skip"
```

**参数说明**：
- `file`: 导入的 JSON 文件
- `conflictStrategy`: 冲突处理策略
  - `skip`: 跳过已存在的记录（推荐）
  - `overwrite`: 覆盖已存在的记录
  - `error`: 遇到冲突时报错

**响应示例**：
```json
{
  "success": true,
  "message": "导入任务已创建",
  "data": {
    "jobId": "import_1234567890_abc123",
    "status": "pending",
    "progress": 0,
    "totalRecords": 100
  }
}
```

#### 查询导入进度
```bash
curl -X GET \
  http://localhost:3000/api/v1/admin/tenants/TENANT_ID/import/JOB_ID \
  -H "Authorization: Bearer YOUR_TOKEN"
```

**响应示例**：
```json
{
  "success": true,
  "data": {
    "jobId": "import_1234567890_abc123",
    "status": "processing",
    "progress": 45,
    "totalRecords": 100,
    "processedRecords": 45,
    "skippedRecords": 5,
    "errorRecords": 0,
    "errors": [],
    "createdAt": "2024-03-03T10:00:00.000Z"
  }
}
```

## 🔄 冲突处理策略

### skip（推荐）
- 跳过已存在的记录
- 继续导入其他数据
- 适合首次导入或增量导入

### overwrite
- 覆盖已存在的记录
- 适合数据同步或更新
- 注意：会丢失现有数据的修改

### error
- 遇到冲突时报错并停止
- 适合严格的数据一致性要求
- 需要手动处理冲突后重新导入

---

## 📊 任务状态

| 状态 | 说明 |
|------|------|
| pending | 任务已创建，等待执行 |
| processing | 正在导入中 |
| completed | 导入完成 |
| failed | 导入失败 |

---

## ⚠️ 注意事项

1. **文件大小**：最大支持 100MB
2. **文件格式**：只支持 JSON 格式
3. **权限要求**：需要 Admin 权限
4. **租户验证**：导入前会验证租户是否存在
5. **数据验证**：会验证数据格式和版本号
6. **错误处理**：导入失败不会回滚已导入的数据

---

## 🔍 故障排查

### 问题：导入任务一直处于 pending 状态
**解决**：检查后端日志，可能是文件读取失败或数据格式错误

### 问题：导入失败，提示"不支持的数据版本"
**解决**：确保导入文件的 version 字段为 "1.0"

### 问题：导入完成但部分数据缺失
**解决**：
1. 查看任务的 `errorRecords` 和 `errors` 字段
2. 检查数据格式是否正确
3. 确认租户 ID 是否匹配

## 📝 完整示例

### JavaScript/Axios
```javascript
const axios = require('axios');
const FormData = require('form-data');
const fs = require('fs');

async function importTenantData(tenantId, filePath, token) {
  // 1. 创建导入任务
  const formData = new FormData();
  formData.append('file', fs.createReadStream(filePath));
  formData.append('conflictStrategy', 'skip');

  const createResponse = await axios.post(
    `http://localhost:3000/api/v1/admin/tenants/${tenantId}/import`,
    formData,
    {
      headers: {
        ...formData.getHeaders(),
        'Authorization': `Bearer ${token}`
      }
    }
  );

  const jobId = createResponse.data.data.jobId;
  console.log('导入任务已创建:', jobId);

  // 2. 轮询查询进度
  while (true) {
    const progressResponse = await axios.get(
      `http://localhost:3000/api/v1/admin/tenants/${tenantId}/import/${jobId}`,
      {
        headers: { 'Authorization': `Bearer ${token}` }
      }
    );

    const job = progressResponse.data.data;
    console.log(`进度: ${job.progress}% | 状态: ${job.status}`);

    if (job.status === 'completed') {
      console.log('✅ 导入完成！');
      console.log(`总记录: ${job.totalRecords}`);
      console.log(`已处理: ${job.processedRecords}`);
      console.log(`跳过: ${job.skippedRecords}`);
      console.log(`错误: ${job.errorRecords}`);
      break;
    } else if (job.status === 'failed') {
      console.error('❌ 导入失败:', job.errors);
      break;
    }

    await new Promise(resolve => setTimeout(resolve, 1000));
  }
}

// 使用示例
importTenantData('tenant-001', './export_data.json', 'YOUR_TOKEN');
```
