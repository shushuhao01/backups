## 🔍 常见问题排查

### 问题 1：测试脚本登录失败
**原因**：Admin 用户不存在或密码错误  
**解决**：
```bash
cd backend
node create-admin-user.js
```

### 问题 2：数据库表不存在
**原因**：未执行数据库迁移  
**解决**：
```bash
mysql -u root -p crm_local < backend/database-migrations/create-tenant-logs-table.sql
```

### 问题 3：导出/导入任务一直 pending
**原因**：后端服务未正常运行  
**解决**：
```bash
cd backend
npm run dev
```

### 问题 4：性能测试未达标
**原因**：未执行性能优化 SQL  
**解决**：
```bash
mysql -u root -p crm_local < backend/database-migrations/optimize-tenant-performance.sql
```

### 问题 5：日志未记录
**原因**：TenantLog 实体未注册  
**解决**：检查 `backend/src/config/database.ts` 中是否包含 TenantLog 实体
