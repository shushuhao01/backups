## ⚠️ 重要注意事项

### 1. 现有用户数据

数据库中现有的用户数据的`tenant_id`字段为`NULL`，这是正确的：
- 私有部署模式：所有用户的`tenant_id`都应该为`NULL`
- SaaS模式：新创建的用户会自动设置`tenant_id`

### 2. 不要直接使用TypeORM Repository

```typescript
// ❌ 错误：不会自动添加租户过滤
const users = await AppDataSource.getRepository(User).find();

// ✅ 正确：使用BaseRepository
const userRepo = new BaseRepository(User);
const users = await userRepo.find();
```
