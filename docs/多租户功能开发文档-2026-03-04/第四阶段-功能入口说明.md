## 三、租户数据导入功能入口

### 1. 后端 API 路由

**基础路径**：`/api/admin/tenants/:id/import`

**API 端点**：

| 方法 | 路径 | 功能 | 说明 |
|------|------|------|------|
| POST | `/api/admin/tenants/:id/import` | 创建导入任务 | 上传 JSON 文件，支持冲突策略 |
| GET | `/api/admin/tenants/:id/import/:jobId` | 查询导入进度 | 实时查看导入状态 |

**请求参数**：
- **file**：JSON 格式文件（multipart/form-data）
- **conflictStrategy**：冲突处理策略
  - `skip`：跳过冲突记录（默认）
  - `overwrite`：覆盖已存在记录
  - `error`：遇到冲突时报错

**请求示例**：
```javascript
const formData = new FormData();
formData.append('file', file);
formData.append('conflictStrategy', 'skip');

await request.post(`/tenants/${tenantId}/import`, formData, {
  headers: { 'Content-Type': 'multipart/form-data' }
});
```

**响应示例**：
```json
{
  "success": true,
  "message": "导入任务已创建",
  "data": {
    "jobId": "import_1234567890",
    "status": "processing",
    "progress": 0,
    "totalRecords": 1500
  }
}
```

- `page`：页码（默认 1）
- `pageSize`：每页数量（默认 20）
