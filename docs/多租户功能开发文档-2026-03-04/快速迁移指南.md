# 快速迁移指南 - CRM Local

## 🚀 最简单的方法

### 方法1：使用一键脚本（推荐）

双击运行 `一键迁移.bat` 文件，按提示输入MySQL密码即可。

脚本会自动完成：
1. ✅ 备份数据库
2. ✅ 执行迁移
3. ✅ 验证结果

---

### 方法2：手动执行（3个命令）

打开 PowerShell 或 CMD，在项目根目录执行：

```bash
# 第1步：备份数据库（输入密码）
mysqldump -u root -p crm_local > D:\kaifa\backup\crm_local_before_migration.sql

# 第2步：执行迁移（输入密码）
mysql -u root -p crm_local < backend/database-migrations/add-tenant-support.sql

# 第3步：验证结果
cd backend
node scripts/migrate-crm-local.js
```

## 📝 注意事项

1. 确保 MySQL 服务已启动
2. 确保数据库名称是 `crm_local`
3. 备份文件会保存到 `D:\kaifa\backup\`
4. 迁移过程中会提示输入 MySQL 密码（2次）

---

## 🎯 迁移完成后

1. 启动后端测试：
```bash
cd backend
npm run dev
```

2. 启动前端测试（新终端）：
```bash
npm run dev
```

3. 测试功能：
   - 登录
   - 查看客户列表
   - 查看订单列表
   - 创建新客户
   - 创建新订单

---

## ⚠️ 如果失败

恢复数据库：
```bash
# 1. 删除数据库
mysql -u root -p -e "DROP DATABASE crm_local;"

# 2. 重新创建
mysql -u root -p -e "CREATE DATABASE crm_local CHARACTER SET utf8mb4 COLLATE utf8mb4_unicode_ci;"

# 3. 恢复备份
mysql -u root -p crm_local < D:\kaifa\backup\crm_local_before_migration.sql
```

---
