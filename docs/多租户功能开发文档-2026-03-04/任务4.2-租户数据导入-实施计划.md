# 任务 4.2：租户数据导入功能

## 📋 任务信息

**优先级**：🔵 低  
**预计时间**：4小时  
**状态**：🚀 进行中

---

## 🎯 目标

实现租户数据的导入功能，支持：
1. 导入 JSON 格式的数据包
2. 验证数据格式和版本
3. 冲突处理策略（跳过/覆盖/报错）
4. 异步导入，提供进度查询

---

## 📦 实施步骤

### 步骤 1：创建导入服务类
**文件**：`backend/src/services/TenantImportService.ts`

**功能**：
- 验证导入文件格式
- 解析 JSON 数据
- 处理数据冲突
- 批量插入数据
- 记录导入日志

### 步骤 2：创建导入 API
**文件**：`backend/src/routes/admin/tenant-import.ts`

**API 列表**：
```
POST   /api/admin/tenants/:id/import        # 创建导入任务
GET    /api/admin/tenants/:id/import/:jobId # 查询导入进度
```

### 步骤 3：实现冲突处理
**策略**：
- skip: 跳过已存在的记录
- overwrite: 覆盖已存在的记录
- error: 遇到冲突时报错

### 步骤 4：测试验证
**测试内容**：
- 导入完整性验证
- 冲突处理测试
- 大数据量导入测试
