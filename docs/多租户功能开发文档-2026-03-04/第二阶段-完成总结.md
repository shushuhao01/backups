## 关键技术点

### 1. 字段命名映射

```typescript
// 数据库：下划线命名
tenant_id VARCHAR(36)

// 实体：驼峰命名
@Column({ name: 'tenant_id' })
tenantId?: string;
```

### 2. 部署模式判断

```typescript
if (deployConfig.isSaaS()) {
  // SaaS模式：使用租户ID
  const tenantId = TenantContextManager.getTenantId();
} else {
  // 私有模式：tenant_id = NULL
  tenantId = null;
}
```

### 3. 自动租户过滤

```typescript
// BaseRepository自动添加过滤
async find(options?: FindManyOptions<Entity>): Promise<Entity[]> {
  options.where = this.addTenantFilter(options.where);
  return this.repository.find(options);
}
```

### 4. 中间件顺序

```typescript
// 正确的顺序
app.use('/api/v1/xxx', tenantAuth, tenantContextMiddleware, xxxRouter);

// 1. tenantAuth: 验证租户身份
// 2. tenantContextMiddleware: 设置租户上下文
// 3. xxxRouter: 业务路由
```
