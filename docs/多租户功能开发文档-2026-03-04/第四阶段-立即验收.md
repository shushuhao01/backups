## 🔍 常见问题

### Q1: 测试报500错误？
**A**: 后端服务未重启，请执行步骤1

### Q2: 测试报404错误？
**A**: 路由未加载，请确认后端服务完全重启

### Q3: 登录失败？
**A**: 执行以下命令创建Admin用户：
```bash
cd backend
node create-admin-user.js
```

### Q4: 数据库连接失败？
**A**: 检查MySQL服务是否运行，数据库crm_local是否存在
