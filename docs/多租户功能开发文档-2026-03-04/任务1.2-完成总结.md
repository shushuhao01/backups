## 🎯 核心功能

### 1. 类型安全的配置存储
```typescript
// 字符串配置
setting.settingType = 'string';
setting.setValue('测试公司名称');
console.log(setting.getValue()); // "测试公司名称"

// 数字配置
setting.settingType = 'number';
setting.setValue(10000);
console.log(setting.getValue()); // 10000 (number类型)

// 布尔配置
setting.settingType = 'boolean';
setting.setValue(true);
console.log(setting.getValue()); // true (boolean类型)

// JSON配置
setting.settingType = 'json';
setting.setValue({ theme: 'dark', color: '#409EFF' });
console.log(setting.getValue()); // { theme: 'dark', color: '#409EFF' }

// 数组配置
setting.settingType = 'array';
setting.setValue(['orders', 'customers', 'products']);
console.log(setting.getValue()); // ['orders', 'customers', 'products']
```

## 📊 表结构详情

### tenant_settings表
```sql
CREATE TABLE `tenant_settings` (
  `id` VARCHAR(36) PRIMARY KEY,
  `tenant_id` VARCHAR(36) NOT NULL,
  `setting_key` VARCHAR(100) NOT NULL,
  `setting_value` TEXT NULL,
  `setting_type` VARCHAR(20) DEFAULT 'string',
  `description` VARCHAR(500) NULL,
  `created_at` DATETIME DEFAULT CURRENT_TIMESTAMP,
  `updated_at` DATETIME DEFAULT CURRENT_TIMESTAMP ON UPDATE CURRENT_TIMESTAMP,
  
  UNIQUE KEY `uk_tenant_setting` (`tenant_id`, `setting_key`),
  KEY `idx_tenant_id` (`tenant_id`),
  FOREIGN KEY (`tenant_id`) REFERENCES `tenants`(`id`)
) ENGINE=InnoDB DEFAULT CHARSET=utf8mb4;
```

### 索引说明
1. **PRIMARY KEY (id)** - 主键索引
2. **uk_tenant_setting (tenant_id, setting_key)** - 复合唯一索引
   - 确保同一租户的配置键唯一
   - 优化按租户和配置键查询
3. **idx_tenant_id (tenant_id)** - 普通索引
   - 优化按租户查询所有配置

## 🔧 使用示例

### 创建配置
```typescript
const setting = new TenantSettings();
setting.id = uuidv4();
setting.tenantId = tenant.id;
setting.settingKey = 'theme_config';
setting.settingType = 'json';
setting.setValue({
  primaryColor: '#409EFF',
  logo: 'https://example.com/logo.png'
});
setting.description = '主题配置';

await manager.save(setting);
```

### 查询配置
```typescript
// 查询租户的所有配置
const settings = await manager.find(TenantSettings, {
  where: { tenantId: tenant.id }
});

// 查询特定配置
const themeSetting = await manager.findOne(TenantSettings, {
  where: {
    tenantId: tenant.id,
    settingKey: 'theme_config'
  }
});

const themeValue = themeSetting.getValue(); // 自动解析为对象
```

### 更新配置
```typescript
themeSetting.setValue({
  primaryColor: '#67C23A',
  logo: 'https://example.com/new-logo.png'
});

await manager.save(themeSetting);
```
