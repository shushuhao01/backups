## 📋 实现功能

### 1. 基本信息展示/编辑
- **信息展示**: 租户名称、代码、状态、联系人、联系电话、联系邮箱、创建时间、更新时间、备注
- **编辑功能**: 点击"编辑"按钮打开对话框，支持修改所有基本信息
- **表单验证**: 必填字段验证、租户代码格式验证

### 2. 授权信息管理
- **授权码显示**: 支持显示/隐藏、一键复制
- **授权状态**: 显示授权状态标签（有效/待激活/已暂停/已吊销）
- **到期时间**: 显示到期时间，即将过期标红提醒
- **激活时间**: 显示首次激活时间
- **操作按钮**:
  - 续期：打开续期对话框
  - 重新生成授权码：确认后生成新授权码
  - 暂停：暂停租户（status=active时显示）
  - 恢复：恢复租户（status=inactive时显示）

### 3. 资源使用情况统计
- **用户数**: 进度条显示已用/总数，根据使用率显示不同颜色
  - <60%: 绿色
  - 60-80%: 橙色
  - >80%: 红色
- **存储空间**: 进度条显示已用/总数，同样根据使用率显示颜色
- **格式化显示**: MB/GB自动转换

### 4. 租户配置管理
- **配置列表**: 表格显示所有配置项
  - 配置键
  - 配置值（JSON格式化显示）
  - 配置类型（string/number/boolean/json/array）
  - 描述
- **操作功能**:
  - 编辑配置：打开编辑对话框
  - 删除配置：确认后删除
  - 新增配置：点击"编辑配置"按钮
- **配置类型支持**:
  - 字符串（string）
  - 数字（number）
  - 布尔值（boolean）
  - JSON对象（json）
  - 数组（array）

### 5. 用户列表
- **用户信息展示**: 用户名、姓名、手机号、邮箱、角色、状态、最后登录时间
- **角色标签**: 管理员、经理、销售、客服、财务
- **状态标签**: 正常/禁用
- **用户数统计**: 显示总用户数

### 6. 对话框功能
- **编辑基本信息对话框**: 完整的表单验证
- **续期对话框**: 显示当前到期时间，选择新的到期时间
- **配置编辑对话框**: 支持添加/编辑配置项
- **授权码生成成功提示**: 显示新生成的授权码，支持复制

## 🔍 关键实现细节

### 1. 资源使用率计算
```typescript
const getUserUsagePercent = () => {
  if (!detail.value.maxUsers) return 0
  return Math.round(((detail.value.userCount || 0) / detail.value.maxUsers) * 100)
}

const getStorageUsagePercent = () => {
  if (!detail.value.maxStorageGb) return 0
  const maxStorageMb = detail.value.maxStorageGb * 1024
  return Math.round(((detail.value.usedStorageMb || 0) / maxStorageMb) * 100)
}
```

### 2. 进度条颜色动态变化
```typescript
const getProgressColor = (percent: number) => {
  if (percent < 60) return '#67c23a'  // 绿色
  if (percent < 80) return '#e6a23c'  // 橙色
  return '#f56c6c'                     // 红色
}
```

### 3. 配置值格式化显示
```typescript
const formatSettingValue = (row: any) => {
  if (!row.settingValue) return '-'
  if (row.settingType === 'json' || row.settingType === 'array') {
    try {
      return JSON.stringify(JSON.parse(row.settingValue), null, 2)
    } catch {
      return row.settingValue
    }
  }
  return row.settingValue
}
```

### 4. 授权码显示/隐藏
```typescript
const maskKey = (key: string) => {
  if (!key) return ''
  const parts = key.split('-')
  if (parts.length >= 4) return `${parts[0]}-${parts[1]}-****-****`
  return key.substring(0, 12) + '****'
}
```
