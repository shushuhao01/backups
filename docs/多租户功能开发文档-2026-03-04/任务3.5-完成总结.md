## 📋 实现功能

### 1. 用户数限制检查 (`checkUserLimit`)
✅ **功能**:
- 在创建用户前检查租户的 `max_users` 限制
- 查询当前租户的实际用户数
- 判断是否超过限制
- 返回友好的错误提示（包含当前用户数、最大用户数、使用率）
- 当使用率≥90%时记录警告日志

✅ **错误响应**:
```json
{
  "success": false,
  "message": "用户数已达上限，当前 10/10 个用户",
  "code": "USER_LIMIT_EXCEEDED",
  "data": {
    "currentCount": 10,
    "maxUsers": 10,
    "usagePercent": 100
  }
}
```

### 2. 存储空间限制检查 (`checkStorageLimit`)
✅ **功能**:
- 在上传文件前检查租户的 `max_storage_gb` 限制
- 支持多种文件大小获取方式：
  - Content-Length请求头
  - multer上传的文件对象
  - 多文件上传
- 计算剩余存储空间
- 判断是否超过限制
- 返回友好的错误提示（包含需要空间、已用空间、剩余空间、使用率）
- 当使用率≥90%时记录警告日志

✅ **错误响应**:
```json
{
  "success": false,
  "message": "存储空间不足，需要 500.00MB，剩余 200.00MB",
  "code": "STORAGE_LIMIT_EXCEEDED",
  "data": {
    "requiredMb": 500.00,
    "usedMb": 800.00,
    "maxMb": 1024,
    "availableMb": 200.00,
    "usagePercent": 78
  }
}
```

### 3. 资源统计更新函数

✅ **updateTenantUserCount(tenantId, increment)**
- 增加租户用户数统计
- 在成功创建用户后调用
- 记录操作日志

✅ **decrementTenantUserCount(tenantId, decrement)**
- 减少租户用户数统计
- 在删除用户后调用
- 确保不会小于0

✅ **updateTenantStorage(tenantId, fileSizeMb)**
- 增加租户存储空间统计
- 在成功上传文件后调用
- 记录操作日志

✅ **decrementTenantStorage(tenantId, fileSizeMb)**
- 减少租户存储空间统计
- 在删除文件后调用
- 确保不会小于0

### 4. 资源使用情况查询

✅ **getTenantResourceUsage(tenantId)**

- 返回详细的错误信息
- 包含当前使用情况和限制
- 提供错误代码便于前端处理

- 便于管理员及时发现资源紧张情况

- 提供增加和减少统计的函数
- 确保统计数据不会小于0
- 记录操作日志便于追踪

## 🔧 使用方法

### 1. 在用户创建路由中使用

```typescript
import { checkUserLimit, updateTenantUserCount } from '../middleware/checkTenantLimits';

router.post('/users', 
  tenantAuth,
  checkUserLimit,  // 检查用户数限制
  async (req, res) => {
    const user = await createUser(req.body);
    await updateTenantUserCount(req.tenantId, 1);  // 更新统计
    res.json({ success: true, data: user });
  }
);
```

### 2. 在文件上传路由中使用

```typescript
import { checkStorageLimit, updateTenantStorage } from '../middleware/checkTenantLimits';

router.post('/upload', 
  tenantAuth,
  checkStorageLimit,  // 检查存储空间限制
  upload.single('file'),
  async (req, res) => {
    const fileSizeMb = req.file.size / (1024 * 1024);
    await updateTenantStorage(req.tenantId, fileSizeMb);  // 更新统计
    res.json({ success: true });
  }
);
```

### 3. 在删除操作中更新统计

```typescript
import { decrementTenantUserCount, decrementTenantStorage } from '../middleware/checkTenantLimits';

// 删除用户
router.delete('/users/:id', async (req, res) => {
  await deleteUser(req.params.id);
  await decrementTenantUserCount(req.tenantId, 1);
  res.json({ success: true });
});

// 删除文件
router.delete('/files/:id', async (req, res) => {
  const file = await getFile(req.params.id);
  await deleteFile(req.params.id);
  await decrementTenantStorage(req.tenantId, file.size / (1024 * 1024));
  res.json({ success: true });
});
```
