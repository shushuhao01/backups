## 🔍 故障排查

### 问题 1：500 错误
**原因**：后端服务未重启，TenantLog 实体未加载  
**解决**：重启后端服务

### 问题 2：404 错误
**原因**：路由未正确注册  
**解决**：检查路由文件是否正确

### 问题 3：登录失败
**原因**：Admin 用户不存在  
**解决**：
```bash
cd backend
node create-admin-user.js
```
