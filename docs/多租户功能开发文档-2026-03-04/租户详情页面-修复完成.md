# 租户详情页面 - 修复完成

## 问题描述
用户反馈：租户详情页面存在大量硬编码数据、字段映射错误、API端点错误等问题。

## 发现的问题

### 1. 字段映射错误
- ❌ 使用 snake_case: `license_key`, `license_status`, `user_count`, `created_at` 等
- ✅ 应使用 camelCase: `licenseKey`, `licenseStatus`, `userCount`, `createdAt` 等

### 2. API端点错误
- ❌ 错误路径: `/admin/tenants/:id`
- ✅ 正确路径: `/tenants/:id`
- 说明：前端 API 配置的 baseURL 已经包含 `/api/v1/admin`

### 3. 续期功能参数错误
- ❌ 发送参数: `{ expireDate: '2027-12-31' }`
- ✅ 正确参数: `{ months: 12 }`
- 说明：后端期望续期月数，而不是具体日期

### 4. 硬编码假数据
- 详情页面在 catch 块中返回硬编码的假数据
- 用户列表也是硬编码的假数据
- 这会误导用户，应该显示真实错误

### 5. 缺少功能
- 缺少存储空间使用进度条
- 缺少用户数使用进度条
- 用户列表未实现真实API调用

```typescript
const confirmRenew = async () => {
  if (!renewExpireDate.value) {
    ElMessage.warning('请选择新的到期时间')
    return
  }
  submitting.value = true
  try {
    // 计算续期月数
    const currentDate = detail.value.expireDate ? new Date(detail.value.expireDate) : new Date()
    const newDate = new Date(renewExpireDate.value)
    const months = Math.max(1, Math.round((newDate.getTime() - currentDate.getTime()) / (30 * 24 * 60 * 60 * 1000)))
    
    const res = await request.post(`/tenants/${route.params.id}/renew`, {
      months
    })
    if (res.success) {
      ElMessage.success('续期成功')
      showRenewDialog.value = false
      fetchDetail()
    }
  } finally {
    submitting.value = false
  }
}
```

```typescript
const formatStorage = (mb: number) => {
```

### 2. 后端修复 (`backend/src/controllers/admin/TenantController.ts`)

#### 统一返回 camelCase 格式
```typescript
export const getTenantDetail = async (req: Request, res: Response): Promise<void> => {
  try {
    const { id } = req.params;
    const tenantRepo = AppDataSource.getRepository(Tenant);
    const tenant = await tenantRepo.findOne({ where: { id } });

    if (!tenant) {
      res.status(404).json({
        success: false,
        message: '租户不存在'
      });
      return;
    }

    // 查询用户数
    const userRepo = AppDataSource.getRepository(User);
    const userCount = await userRepo.count({ where: { tenantId: id } });

    // 转换为 camelCase 格式返回
    res.json({
      success: true,
      data: {
        id: tenant.id,
        name: tenant.name,
        code: tenant.code,
        packageId: tenant.packageId,
        packageName: null, // TODO: 从packages表查询
        contact: tenant.contact,
        phone: tenant.phone,
        email: tenant.email,
        maxUsers: tenant.maxUsers,
        maxStorageGb: tenant.maxStorageGb,
        userCount,
        usedStorageMb: tenant.usedStorageMb,
        expireDate: tenant.expireDate ? (tenant.expireDate instanceof Date ? tenant.expireDate.toISOString().split('T')[0] : tenant.expireDate) : null,
        licenseKey: tenant.licenseKey,
        licenseStatus: tenant.licenseStatus,
        activatedAt: tenant.activatedAt,
        features: tenant.features,
        databaseName: tenant.databaseName,
        remark: tenant.remark,
        status: tenant.status,
        createdAt: tenant.createdAt,
        updatedAt: tenant.updatedAt,
        isExpired: tenant.isExpired(),
        isAvailable: tenant.isAvailable(),
        settings: []
      }
    });
  } catch (error) {
    console.error('获取租户详情失败:', error);
    res.status(500).json({
      success: false,
      message: '获取租户详情失败'
    });
  }
};
```
