**核心功能**：

#### 2.1 自动检测tenant_id字段
```typescript
protected hasTenantIdField(): boolean {
  const metadata = this.repository.metadata;
  return metadata.columns.some(column => 
    column.propertyName === 'tenantId' || column.propertyName === 'tenant_id'
  );
}
```

#### 2.2 获取租户字段属性名
```typescript
protected getTenantFieldName(): string | null {
  const metadata = this.repository.metadata;
  const tenantColumn = metadata.columns.find(column => 
    column.propertyName === 'tenantId' || column.propertyName === 'tenant_id'
  );
  return tenantColumn ? tenantColumn.propertyName : null;
}
```

#### 2.3 自动添加租户过滤
```typescript
protected addTenantFilter(where) {
  const tenantFieldName = this.getTenantFieldName();
  let tenantValue = deployConfig.isSaaS() 
    ? this.getTenantId() 
    : null;
  
  return { ...where, [tenantFieldName]: tenantValue };
}
```

#### 2.4 自动设置tenant_id
```typescript
protected setTenantId(entity) {
  const tenantFieldName = this.getTenantFieldName();
  if (deployConfig.isSaaS()) {
    entity[tenantFieldName] = this.getTenantId();
  } else {
    entity[tenantFieldName] = null;
  }
}
```

## 🎓 技术要点

### 1. AsyncLocalStorage的优势

- 不需要在每个函数中传递tenant_id参数
- 支持异步调用链中传递上下文
- 线程安全，不会混淆不同请求的租户信息

### 2. TypeORM元数据的使用

- 通过`repository.metadata.columns`获取实体字段信息
- 动态检测字段名，支持不同命名风格
- 获取数据库列名用于QueryBuilder

### 3. 部署模式兼容

- SaaS模式：`WHERE tenant_id = 'xxx'`
- 私有模式：`WHERE tenant_id IS NULL`
- 通过`deployConfig.isSaaS()`动态判断

### 4. 类型安全

- 使用TypeScript泛型保持类型安全
- 方法签名与TypeORM Repository一致
- 支持方法重载（如save方法）

## ⚠️ 注意事项

### 1. 实体命名规范

**必须遵守**：
- 数据库字段：`tenant_id`（下划线命名）
- 实体属性：`tenantId`（驼峰命名）
- 映射方式：`@Column({ name: 'tenant_id' })`

### 2. 使用getRawRepository的风险

```typescript
// ⚠️ 危险：不会自动添加租户过滤
const rawRepo = customerRepo.getRawRepository();
const customers = await rawRepo.find(); // 会查询所有租户的数据！

// ✅ 安全：使用BaseRepository
const customers = await customerRepo.find(); // 自动过滤
```

### 3. 没有tenant_id字段的表

BaseRepository会自动检测，如果表没有tenant_id字段，不会添加过滤：
```typescript
// tenants表本身没有tenant_id字段
const tenantRepo = new BaseRepository(Tenant);
const tenants = await tenantRepo.find(); // 不会添加tenant_id过滤
```

### 4. 复杂查询使用QueryBuilder

```typescript
const qb = customerRepo.createQueryBuilder('customer');
qb.leftJoinAndSelect('customer.orders', 'order')
  .where('customer.status = :status', { status: 'active' });
// tenant_id过滤已自动添加到customer表

const customers = await qb.getMany();
```

import { BaseRepository } from '../repositories/BaseRepository';

export class CustomerService {
  private customerRepo: BaseRepository<Customer>;

    this.customerRepo = new BaseRepository(Customer);

    // 自动添加tenant_id过滤
    await this.customerRepo.update(id, data);
  }

    // 自动添加tenant_id过滤
    await this.customerRepo.delete(id);
  }
}
