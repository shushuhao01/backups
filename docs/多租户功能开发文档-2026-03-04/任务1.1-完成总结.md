## 🔧 技术细节

### TypeORM装饰器使用
```typescript
@Entity('tenants')                    // 映射到tenants表
@PrimaryColumn('varchar', {})         // 主键
@Column('varchar', {})                // 普通列
@Index('idx_code', { unique: true })  // 唯一索引
@CreateDateColumn()                   // 自动创建时间
@UpdateDateColumn()                   // 自动更新时间
```

### 字段类型映射
- VARCHAR → string
- INT → number
- DECIMAL(10,2) → number
- DATE → Date
- DATETIME → Date
- JSON → any
- TEXT → string
