# 租户客户列表 - 快速参考

## 主要优化

### 1. 操作列（4个按钮 + 更多）
- 详情
- 编辑  
- 暂停/恢复
- **更多** ▼
  - 续期
  - 重新生成授权码
  - 删除

### 2. 授权码（图标操作）
- 👁️ 显示/隐藏
- 📋 复制

### 3. 租户编码（自动生成）
格式：`T260303A1B2`
- T = Tenant
- 260303 = 2026年3月3日
- A1B2 = 随机字符

### 4. 列调整
- ❌ 移除：联系电话
- ✅ 新增：存储空间（已用/总量）

### 5. 列宽优化
所有列都不会换行，显示更紧凑
