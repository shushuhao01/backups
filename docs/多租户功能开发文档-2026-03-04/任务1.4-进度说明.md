## 💡 技术要点

### 1. AsyncLocalStorage的使用

```typescript
const asyncLocalStorage = new AsyncLocalStorage<TenantContext>();

// 在中间件中设置上下文
asyncLocalStorage.run(context, () => {
  next();
});

// 在任何地方获取上下文
const tenantId = asyncLocalStorage.getStore()?.tenantId;
```

### 2. 动态字段名检测

```typescript
const metadata = this.repository.metadata;
const tenantColumn = metadata.columns.find(
  col => col.propertyName === 'tenant_id' || col.propertyName === 'tenantId'
);

if (tenantColumn) {
  const fieldName = tenantColumn.propertyName;
  entity[fieldName] = tenantId;
}
```

### 3. 租户过滤条件生成

```typescript
protected addTenantFilter(where) {
  const tenantColumn = this.getTenantColumn();
  const tenantValue = this.getTenantValue();
  
  return {
    ...where,
    [tenantColumn.propertyName]: tenantValue
  };
}
```
