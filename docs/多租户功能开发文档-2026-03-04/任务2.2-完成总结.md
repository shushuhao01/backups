#### 2.1 租户验证逻辑
```typescript
// 租户验证（SaaS模式）
let tenantId: string | undefined;
let tenantInfo: Tenant | null = null;

if (deployConfig.isSaaS()) {
  // SaaS模式：必须提供license_key
  if (!license_key) {
    throw new BusinessError('SaaS模式下必须提供授权码', 'LICENSE_KEY_REQUIRED');
  }

  // 验证license_key，查询租户
  const tenant = await tenantRepository.findOne({
    where: { licenseKey: license_key }
  });

  if (!tenant) {
    throw new BusinessError('授权码无效', 'INVALID_LICENSE_KEY');
  }

  // 检查租户状态
  if (tenant.status !== 'active') {
    throw new BusinessError('租户已被禁用，请联系管理员', 'TENANT_DISABLED');
  }

  // 检查授权状态
  if (tenant.licenseStatus !== 'active') {
    throw new BusinessError('租户授权状态异常', 'TENANT_LICENSE_INVALID');
  }

  // 检查过期时间
  if (tenant.expireDate && new Date(tenant.expireDate) < new Date()) {
    throw new BusinessError('租户已过期，请联系管理员续费', 'TENANT_EXPIRED');
  }

  tenantId = tenant.id;
  tenantInfo = tenant;
} else {
  // 私有部署模式：不需要license_key
  tenantId = undefined;
  tenantInfo = null;
}
```

#### 2.2 用户查询（添加租户过滤）
```typescript
// 查找用户（添加租户过滤）
const user = await this.userRepository.findOne({
  where: {
    username,
    tenantId: tenantId || null  // SaaS模式：匹配租户ID，私有模式：匹配NULL
  }
});
```

#### 2.3 JWT Token生成（包含tenantId）
```typescript
// 生成JWT令牌（包含tenantId）
const tokenPayload = {
  userId: user.id,
  username: user.username,
  role: user.roleId || user.role,
  departmentId: user.departmentId,
  tenantId: tenantId  // SaaS模式有值，私有模式为undefined
};
```

#### 2.4 返回租户信息

```typescript
res.json({
```

## 📊 功能特性

### 1. 部署模式兼容

| 模式 | license_key | 租户验证 | JWT Token | 说明 |
|------|------------|---------|-----------|------|
| 私有部署 | 可选（忽略） | 不验证 | 不包含tenantId | 向后兼容 |
| SaaS | 必填 | 验证租户 | 包含tenantId | 新功能 |

### 2. 租户验证流程

**SaaS模式**：
1. 检查license_key是否提供
2. 根据license_key查询租户
3. 验证租户状态（active）
4. 验证授权状态（active）
5. 验证过期时间
6. 查询用户（带租户过滤）
7. 验证密码
8. 生成JWT Token（包含tenantId）
9. 返回用户信息和租户信息

**私有部署模式**：
1. 查询用户（tenant_id = NULL）
2. 验证密码
3. 生成JWT Token（不包含tenantId）
4. 返回用户信息

### 3. 错误处理

| 错误场景 | 错误码 | 错误信息 |
|---------|--------|---------|
| 缺少license_key（SaaS） | LICENSE_KEY_REQUIRED | SaaS模式下必须提供授权码 |
| license_key无效 | INVALID_LICENSE_KEY | 授权码无效 |
| 租户已禁用 | TENANT_DISABLED | 租户已被禁用，请联系管理员 |
| 授权待激活 | TENANT_LICENSE_INVALID | 租户授权待激活，请联系管理员 |
| 授权已暂停 | TENANT_LICENSE_INVALID | 租户授权已暂停，请联系管理员 |
| 授权已过期 | TENANT_LICENSE_INVALID | 租户授权已过期，请联系管理员续费 |
| 租户已过期 | TENANT_EXPIRED | 租户已过期，请联系管理员续费 |
| 用户名或密码错误 | INVALID_CREDENTIALS | 用户名或密码错误 |
| 账户已锁定 | ACCOUNT_LOCKED | 账户已被锁定，请联系管理员 |
| 账户已禁用 | ACCOUNT_DISABLED | 账户已被禁用，请联系管理员 |
| IP未授权 | IP_NOT_AUTHORIZED | 当前IP地址未授权登录，请联系管理员 |


- 不需要提供license_key
