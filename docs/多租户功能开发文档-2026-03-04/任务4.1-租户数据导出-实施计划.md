# 任务 4.1：租户数据导出功能

## 📋 任务信息

**优先级**：🔵 低  
**预计时间**：4小时  
**状态**：🚀 进行中

---

## 🎯 目标

实现租户数据的完整导出功能，支持：
1. 导出指定租户的所有业务数据
2. 选择导出的数据范围
3. 异步导出，提供进度查询
4. 生成 JSON 格式的数据包

---

## 📦 实施步骤

### 步骤 1：创建导出服务类
**文件**：`backend/src/services/TenantExportService.ts`

**功能**：
- 查询租户的所有业务数据
- 按表分类组织数据
- 生成 JSON 格式的导出文件
- 支持增量导出（按时间范围）

### 步骤 2：创建导出 API
**文件**：`backend/src/routes/admin/tenant-export.ts`

**API 列表**：
```
POST   /api/admin/tenants/:id/export        # 创建导出任务
GET    /api/admin/tenants/:id/export/:jobId # 查询导出进度
GET    /api/admin/tenants/:id/export/:jobId/download # 下载导出文件
```

### 步骤 3：实现异步导出
**功能**：
- 使用后台任务处理导出
- 记录导出进度
- 生成下载链接

### 步骤 4：测试验证
**测试内容**：
- 导出完整性验证
- 大数据量导出测试
- 并发导出测试
