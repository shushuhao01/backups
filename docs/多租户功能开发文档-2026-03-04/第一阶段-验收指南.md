## 🐛 常见问题

### 问题1：测试脚本报错 "Cannot find module"

**原因**：TypeScript未编译或编译失败

**解决方案**：
```bash
cd backend
npm run build
```

### 问题2：数据库连接失败

**原因**：数据库配置不正确或数据库未启动

**解决方案**：
1. 检查 `backend/.env.local` 配置
2. 确保MySQL数据库已启动
3. 确保数据库名称、用户名、密码正确

### 问题3：测试数据冲突

**原因**：之前的测试数据未清理

**解决方案**：
```sql
-- 清理测试数据
DELETE FROM tenant_settings WHERE tenant_id LIKE 'test-tenant-%';
DELETE FROM tenants WHERE id LIKE 'test-tenant-%';
DELETE FROM users WHERE username LIKE 'test_user_%';
```

### 问题4：端口被占用

**原因**：其他进程占用了数据库端口

**解决方案**：
1. 检查端口占用：`netstat -ano | findstr :3306`
2. 关闭占用端口的进程
3. 或修改数据库端口配置
