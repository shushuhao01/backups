## Admin后台登录问题

Admin后台登录提示"API端点不存在"的问题，我已经定位到原因：

在 `backend/src/routes/admin/index.ts` 中，systemConfig路由注册方式有误：

```typescript
// 错误的写法（会拦截所有路径）
router.use('/', systemConfigRouter);

// 正确的写法
router.use(systemConfigRouter);
```

**修复已完成**，但需要重启后端服务才能生效。
