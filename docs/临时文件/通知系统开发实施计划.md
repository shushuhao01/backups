## 技术要点

### 1. 网站地址使用
所有通知中的链接都使用配置的地址:
```typescript
import { SITE_CONFIG } from '@/config/sites'

const loginUrl = `${SITE_CONFIG.CRM_URL}/login`
const registerUrl = `${SITE_CONFIG.WEBSITE_URL}/register`
const renewUrl = `${SITE_CONFIG.WEBSITE_URL}/renew`
```

### 2. 模板变量替换
```typescript
// 模板内容
"欢迎注册{{systemName}}，您的账号是{{adminUsername}}"

// 变量数据
{ systemName: '云客CRM', adminUsername: 'admin' }

// 渲染结果
"欢迎注册云客CRM，您的账号是admin"
```

### 3. 多渠道发送
```typescript
// 自动根据模板配置发送到多个渠道
await notificationTemplateService.sendByTemplate('template_code', variables)
// 会自动发送: 邮件 + 短信 + 微信(如果用户已绑定)
```

### 4. 微信消息推送
```typescript
// 查找租户绑定的微信用户
const followers = await wechatFollowerService.getByTenantId(tenantId)

// 推送模板消息
for (const follower of followers) {
  await wechatMessageService.sendTemplateMessage(
    follower.openid,
    templateCode,
    data
  )
}
```
