# 同步命令 - 超简单版 ⚡

## 🎯 你只需要记住这4条命令

在IDE底部的终端，依次输入:

```powershell
git add .
git commit -m "更新"
git push origin main
.\sync-repos-v2.ps1 -message "更新"
```

**就这4条!** 每次都一样!

---

## 💡 如果你想写详细点

把 `"更新"` 改成你做了什么:

```powershell
git add .
git commit -m "修复订单bug"
git push origin main
.\sync-repos-v2.ps1 -message "修复订单bug"
```

**重点**: 两个地方写的要一样!

---

## 🚀 更快的方法（一次性执行）

复制这条命令，粘贴到终端:

```powershell
git add . ; git commit -m "更新" ; git push origin main ; .\sync-repos-v2.ps1 -message "更新"
```

按回车，全部自动执行!

---

## 📍 在哪里输入？

在 **IDE底部的终端窗口**，就是显示 `PS D:\kaifa\CRM - 1.8.0>` 的地方。

---

## ❓ 提交信息写什么？

随便写!比如:
- "更新"
- "修改"
- "fix"
- "优化订单"
- "添加功能"

**都可以!** 只要两个地方写的一样就行。

---

