## 常见问题

### Q1: 手机号格式验证
- **规则**：必须是11位，以1开头，第二位是3-9
- **示例**：13800138000, 15912345678
- **错误示例**：12345678901, 10012345678

### Q2: 授权码前缀
- **租户客户**：TENANT-XXXX-XXXX-XXXX-XXXX
- **私有客户**：PRIVATE-XXXX-XXXX-XXXX-XXXX

### Q3: 默认密码
- **密码**：Aa123456
- **说明**：符合复杂度要求（大写+小写+数字）
- **建议**：首次登录后立即修改

### Q4: 管理员账号
- **用户名**：使用联系电话作为用户名
- **角色**：admin（超级管理员）
- **状态**：active（激活状态）

## 如果遇到问题

### 检查后端日志
查看终端中的后端日志，搜索以下关键字：
- `[Admin Tenants]` - 租户创建相关日志
- `[AdminAccountHelper]` - 管理员账号创建日志
- `ERROR` - 错误信息

### 检查数据库
```sql
-- 检查 salt 字段是否存在
DESCRIBE users;

-- 检查最新创建的租户
SELECT * FROM tenants ORDER BY created_at DESC LIMIT 5;

-- 检查最新创建的用户
SELECT id, username, phone, role, status, created_at 
FROM users 
ORDER BY created_at DESC 
LIMIT 5;
```
