## 💳 模拟支付成功

由于开发环境无法真实支付，需要手动模拟：

### 方法1: 使用模拟脚本（推荐）

```bash
# 1. 在支付页面记录订单号（例如：PAY1772697489638D0Z9ZL）

# 2. 打开新终端，运行模拟脚本
cd backend
node simulate-payment-success.js

# 3. 输入订单号，确认操作

# 4. 返回浏览器，等待自动跳转（3秒内）
```

### 方法2: 直接执行SQL

```bash
cd backend
node -e "const mysql = require('mysql2/promise'); require('dotenv').config({ path: '.env.local' }); (async () => { const conn = await mysql.createConnection({ host: process.env.DB_HOST, port: process.env.DB_PORT, user: process.env.DB_USERNAME, password: process.env.DB_PASSWORD, database: process.env.DB_DATABASE }); await conn.query('UPDATE payment_orders SET status = ?, paid_at = NOW() WHERE order_no = ?', ['paid', '你的订单号']); await conn.query('UPDATE tenants SET status = ?, license_status = ?, activated_at = NOW(), expire_date = DATE_ADD(CURDATE(), INTERVAL 30 DAY) WHERE id = (SELECT tenant_id FROM payment_orders WHERE order_no = ?)', ['active', 'active', '你的订单号']); console.log('✓ 支付状态已更新'); await conn.end(); })()"
```

**注意**: 将"你的订单号"替换为实际订单号
