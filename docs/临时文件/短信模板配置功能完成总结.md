## 12个短信模板场景

| 模板类型 | 字段名 | 使用场景 | 变量 |
|---------|--------|---------|------|
| 验证码模板 | VERIFY_CODE | 注册、登录验证码 | code |
| 注册成功通知 | REGISTER_SUCCESS | 官网注册成功 | tenant_name, license_code, admin_account, admin_password |
| 支付成功通知 | PAYMENT_SUCCESS | 支付成功后 | tenant_name, license_code, admin_account, admin_password |
| 续期成功通知 | RENEW_SUCCESS | 管理员续期 | tenant_name, expire_date |
| 套餐变更通知 | PACKAGE_CHANGE | 升级/降级套餐 | tenant_name, old_package, new_package |
| 配额变更通知 | QUOTA_CHANGE | 调整用户数/存储 | tenant_name, quota_type, old_value, new_value |
| 账号暂停通知 | ACCOUNT_SUSPEND | 暂停授权 | tenant_name, reason |
| 账号激活通知 | ACCOUNT_RESUME | 恢复授权 | tenant_name |
| 账号注销通知 | ACCOUNT_CANCEL | 注销租户 | tenant_name |
| 退款成功通知 | REFUND_SUCCESS | 处理退款 | tenant_name, amount |
| 到期提醒通知 | EXPIRE_REMIND | 到期前7/3/1天 | tenant_name, days_left |
| 过期通知 | EXPIRED_NOTICE | 账号过期当天 | tenant_name |

## 使用指南

### 1. 申请阿里云短信模板
参考文档：`docs/阿里云短信模板申请清单.md`

### 2. 配置到系统
1. 登录Admin后台
2. 进入"系统设置" → "基础配置" → "短信配置"
3. 填写基础配置（AccessKey、Secret、签名）
4. 逐个填写已审核通过的模板CODE
5. 点击"保存配置"

### 3. 测试短信
1. 确保已配置基础信息和验证码模板CODE
2. 点击"发送测试短信"按钮
3. 输入测试手机号
4. 查收短信验证功能

### 4. 生产环境部署
1. 在宝塔phpMyAdmin中执行：`backend/database-migrations/production-add-sms-templates.sql`
2. 在Admin后台配置短信信息
3. 测试验证功能正常
