# 阿里云短信模板 - 快速复制版

## 📋 使用说明

1. 直接复制下面的模板内容到阿里云控制台
2. 不要修改任何内容，直接粘贴
3. 按照变量属性表选择每个变量的类型
4. 提交审核

---

## 1️⃣ 注册验证码（验证码类型）

**模板名称：** 注册验证码

**模板类型：** 验证码

**模板内容：**
```
您的注册验证码是：${code}，5分钟内有效，请勿泄露给他人。
```

**变量属性：**
- `code` → 仅数字

---

## 2️⃣ 账号开通通知（免费试用）

**模板名称：** 账号开通通知

**模板类型：** 短信通知

**模板内容：**
```
尊敬的${tenantName}，您的云客CRM账号已开通！租户编码：${tenantCode}，授权码：${licenseKey}，管理员账号为您的手机号，初始密码：${adminPassword}，套餐：${packageName}，到期日期：${expireDate}。请妥善保管账号信息，登录后请立即修改密码。
```

**变量属性：**
- `tenantName` → 企业/组织名称
- `tenantCode` → 其他（姓名、账号、地址等）
- `licenseKey` → 其他（姓名、账号、地址等）
- `adminPassword` → 其他（姓名、账号、地址等）
- `packageName` → 其他（姓名、账号、地址等）
- `expireDate` → 时间

**使用场景：** 免费试用套餐注册成功后立即发送

## 📝 申请注意事项

1. ✅ 直接复制模板内容，不要修改
2. ✅ 不要在模板开头添加【】签名，系统会自动添加
3. ✅ 按照变量属性表准确选择每个变量的类型
4. ✅ 在"申请说明"中简要描述模板用途
5. ❌ 不要将URL作为变量
6. ❌ 不要将手机号作为变量

## 🎯 优先申请顺序

建议按以下顺序申请（从高到低）：

1. 🔴 注册验证码（必需 - 所有注册都需要）
2. 🔴 账号开通通知（必需 - 免费试用注册成功）
3. 🔴 支付成功账号开通通知（必需 - 付费套餐支付成功）
4. 🔴 账号到期提醒（必需 - 定时任务）
5. 🔴 账号过期通知（必需 - 定时任务）
6. 🟡 续费成功通知（老客户续费）
7. 🟡 套餐变更通知（管理员操作）
8. 🟡 账号暂停通知（管理员操作）
9. 🟡 账号激活通知（管理员操作）
10. 🟢 配额变更通知（管理员操作）
11. 🟢 账号注销通知（管理员操作）
12. 🟢 退款成功通知（退款流程）
