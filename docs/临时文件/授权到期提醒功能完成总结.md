## 一、功能概述

实现了自动化的授权到期检测和提醒系统,包括:

1. **定时检测** - 每天自动检查即将到期的授权
2. **多时间点提醒** - 7天、3天、1天前分别提醒
3. **自动处理到期** - 自动更新已到期授权的状态
4. **通知发送** - 通过邮件/短信发送提醒
5. **手动触发** - 支持Admin后台手动触发检查
6. **统计报表** - 提供到期统计数据

## 二、核心组件

### 2.1 授权到期提醒服务

**文件**: `backend/src/services/LicenseExpirationReminderService.ts`

**功能**:
- `checkAndSendReminders()` - 检查并发送即将到期提醒
- `checkExpiringLicenses(days)` - 查询N天后到期的授权
- `sendExpirationReminder()` - 发送到期提醒通知
- `checkAndHandleExpiredLicenses()` - 检查并处理已到期授权
- `runFullCheck()` - 执行完整检查流程

**检测逻辑**:
```typescript
// 检查7天、3天、1天后到期的授权
const reminderDays = [7, 3, 1];

for (const days of reminderDays) {
  // 查询即将到期的租户
  const tenants = await AppDataSource.query(`
    SELECT * FROM tenants 
    WHERE status = 'active'
      AND DATE(expire_date) = DATE_ADD(CURDATE(), INTERVAL ? DAY)
      AND license_status = 'active'
  `, [days]);
  
  // 发送提醒
  for (const tenant of tenants) {
    await sendExpirationReminder(tenant, days);
  }
}
```

**防重复机制**:
- 检查当天是否已发送过提醒
- 避免同一租户收到重复通知

### 2.2 定时任务调度服务

**文件**: `backend/src/services/SchedulerService.ts`

**功能**:
- `start()` - 启动所有定时任务
- `stop()` - 停止所有定时任务
- `triggerTask(name)` - 手动触发指定任务
- `getTasksStatus()` - 获取所有任务状态

**调度配置**:
```typescript
// 每天早上9点执行授权到期检查
this.scheduleTask(
  'license-expiration-check',
  '0 9 * * *', // Cron表达式
  async () => {
    await licenseExpirationReminderService.runFullCheck();
  },
  '授权到期检查'
);
```

## 三、通知模板

**模板代码**: `license_expire_soon`

**触发时机**: 到期前7天、3天、1天

## 四、执行流程

### 4.1 自动执行流程

```

    ↓
定时任务触发
    ↓
检查7天后到期 ──→ 发送提醒 ──→ 记录日志
    ↓
检查3天后到期 ──→ 发送提醒 ──→ 记录日志
    ↓
检查1天后到期 ──→ 发送提醒 ──→ 记录日志
    ↓
检查已到期授权 ──→ 更新状态 ──→ 发送通知
    ↓
完成
```

```

    ↓

    ↓

    ↓

```

## 九、常见问题

### Q1: 定时任务没有执行?

A: 检查以下几点:
1. 确认后端服务正常运行
2. 查看日志确认调度器已启动
3. 检查系统时间是否正确
4. 尝试手动触发任务测试

### Q2: 通知没有发送?

A: 检查:
1. 邮件/短信服务是否配置正确
2. 租户是否有邮箱或手机号
3. 查看notification_logs表的错误信息
4. 检查是否已发送过(防重复机制)

### Q3: 如何修改检查时间?

A: 修改 `SchedulerService.ts` 中的cron表达式:
```typescript
// 改为每天下午2点执行
'0 14 * * *'

// 改为每小时执行(测试用)
'0 * * * *'
```

### Q4: 如何添加更多提醒时间点?

A: 修改 `LicenseExpirationReminderService.ts`:
```typescript
// 添加15天、30天提醒
const reminderDays = [30, 15, 7, 3, 1];
```
