# 个人业绩客户明细权限修复 - 快速参考

**修复时间**: 2026-03-05  
**问题**: 个人业绩页面客户明细没有按角色过滤,所有人都能看到所有客户  
**修复**: 添加 `onlyMine` 参数,强制只返回当前用户的客户

## 📝 修改文件

### 1. 后端API

**文件**: `backend/src/routes/customers.ts`

**修改**: 添加 `onlyMine` 参数处理逻辑

