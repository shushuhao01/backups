# 授权码验证功能 - 快速参考卡

## 🎯 核心特性

### 授权码类型
- **私有部署**: `PRIVATE-XXXX-XXXX-XXXX-XXXX`
- **SaaS租户**: `TENANT-XXXX-XXXX-XXXX-XXXX` 或 `LIC-XXXX`

### 租户编码
- **私有部署**: `P260321ABCD` (P前缀)
- **SaaS租户**: `T20260XXXXX` (T前缀)

---

## 🔄 验证流程

### 私有部署
```
输入 PRIVATE- 授权码
  ↓
验证 licenses 表
  ↓
创建/更新本地 tenants 记录
  ↓
返回 deployType: 'private'
  ↓
自动保存到 localStorage
```

### SaaS模式
```
输入 TENANT- 授权码
  ↓
验证 tenants 表
  ↓
激活租户 (pending → active)
  ↓
返回 deployType: 'saas'
  ↓
自动保存到 localStorage
```

## 🔑 关键API

### 1. 授权码验证
```
POST /api/v1/tenant-license/verify
Body: { licenseKey: 'PRIVATE-...' | 'TENANT-...' }
Response: { data: { ..., deployType: 'private' | 'saas' } }
```

### 2. 租户编码验证
```
POST /api/v1/tenant-license/verify-code
Body: { tenantCode: 'P260321ABCD' | 'T20260XXXXX' }
Response: { data: { tenantId, tenantName, ... } }
```

### 3. 私有部署检查
```
GET /api/v1/tenant-license/check-private
Response: { data: { activated: true/false, ... } }
```

### 4. 心跳检测
```
POST /api/v1/tenant-license/heartbeat
Response: { valid: true/false }
```

## 🎨 前端使用

### 获取部署模式
```typescript
import { getDeployMode, isPrivateMode } from '@/api/tenantLicense'

const mode = getDeployMode()  // 'private' | 'saas'
const isPrivate = isPrivateMode()  // boolean
```

### 保存部署模式
```typescript
import { saveDeployMode } from '@/api/tenantLicense'

// 激活时自动调用
if (res.data.deployType) {
  saveDeployMode(res.data.deployType)
}
```

## 🗄️ 数据库表

### licenses (私有客户授权)
```sql
license_key: PRIVATE-XXXX-XXXX-XXXX-XXXX
customer_type: 'private'
status: 'pending' | 'active' | 'revoked'
```

### tenants (统一租户表)
```sql
code: P260321ABCD (私有) | T20260XXXXX (SaaS)
license_key: PRIVATE-... | TENANT-...
license_status: 'pending' | 'active' | 'expired' | 'suspended'
```

## 🚀 部署配置

### 后端环境变量
```bash
# 可选：默认为 private
DEPLOY_MODE=private  # 或 saas

# 私有部署配置（可选）
LICENSE_KEY=PRIVATE-XXXX-XXXX-XXXX-XXXX
LICENSE_SERVER=https://admin.yourdomain.com
```

### 前端环境变量
```bash
# 可选：默认为 saas
VITE_DEPLOY_MODE=private  # 或 saas
```

**注意**: 即使不配置环境变量，系统也会根据授权码类型自动识别模式！
