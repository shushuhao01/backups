# 生产环境用户管理500错误 - 快速修复

**问题**: 用户管理页面报500错误  
**原因**: 生产环境 `roles` 表缺少字段  
**修复**: 执行SQL添加字段

---

## 🚀 快速修复(3步)

### 1️⃣ 登录宝塔

访问宝塔面板 > 数据库 > 选择生产数据库 > 管理 > SQL

### 2️⃣ 执行SQL

复制以下SQL并执行:

```sql
-- 添加缺失字段
ALTER TABLE `roles`
  ADD COLUMN `code` varchar(50) NOT NULL DEFAULT '' AFTER `name`,
  ADD COLUMN `status` varchar(20) NOT NULL DEFAULT 'active' AFTER `description`,
  ADD COLUMN `level` int(11) NOT NULL DEFAULT 0 AFTER `status`,
  ADD COLUMN `color` varchar(20) DEFAULT NULL AFTER `level`,
  ADD COLUMN `permissions` json DEFAULT NULL AFTER `color`,
  ADD COLUMN `roleType` varchar(20) DEFAULT NULL AFTER `permissions`,
  ADD COLUMN `data_scope` varchar(20) DEFAULT NULL AFTER `roleType`;

-- 添加索引
ALTER TABLE `roles`
  ADD UNIQUE KEY `uk_code` (`code`),
  ADD KEY `idx_status` (`status`);

-- 更新现有数据
UPDATE `roles` SET `code` = `name` WHERE `code` = '';
UPDATE `roles` SET `roleType` = IF(`is_system` = 1, 'system', 'custom');
UPDATE `roles` SET `data_scope` = 'all';
```

### 3️⃣ 验证修复

刷新浏览器,访问用户管理页面,检查是否正常。

---

## ❌ 缺失的字段

生产环境 `roles` 表缺少:
- `code` - 角色代码
- `status` - 状态
- `level` - 角色级别
- `color` - 角色颜色
- `permissions` - 权限列表
- `roleType` - 角色类型
- `data_scope` - 数据范围
