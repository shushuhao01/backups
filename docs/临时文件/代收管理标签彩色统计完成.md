# 代收管理标签彩色统计完成

## 修改时间
2026-03-24

## 需求说明
为代收管理页面的标签选项卡添加彩色数字统计：
- 待处理：红色数字
- 已返款：绿色数字
- 已改代收：橙色数字
- 无需代收：灰色数字
- 全部：灰色数字

**4. 数据加载**
```typescript
const loadStats = async () => { 
  try { 
    const p: any = {}
    if (startDate.value) p.startDate = startDate.value
    if (endDate.value) p.endDate = endDate.value
    if (departmentFilter.value) p.departmentId = departmentFilter.value
```

### 2. 后端修改

#### 文件

#### 统计逻辑

**1. 待处理数量（pending）**
```typescript
const pendingCount = await orderRepo.count({
  where: { ...tabStatsWhere, codStatus: 'pending' }
});
```

**2. 已返款数量（returned）**
```typescript
const returnedCount = await orderRepo.count({
  where: { ...tabStatsWhere, codStatus: 'returned' }
});
```

**3. 已改代收数量（cancelled）**
```typescript
const cancelledCount = await orderRepo.count({
  where: { ...tabStatsWhere, codStatus: 'cancelled' }
});
```

**4. 无需代收数量（zero）**
```typescript
// 查询所有订单
const allOrders = await orderRepo.find({
  where: tabStatsWhere,
  select: ['id', 'totalAmount', 'depositAmount']
});

// 过滤出原始代收金额为0的订单
const zeroCount = allOrders.filter(o => {
  const originalCodAmount = (Number(o.totalAmount) || 0) - (Number(o.depositAmount) || 0);
  return originalCodAmount === 0;
}).length;
```

**5. 全部数量（all）**
```typescript
const allCount = await orderRepo.count({ where: tabStatsWhere });
```

## 显示规则

1. **数字显示**
   - 当数量 > 0 时，显示数字徽章
   - 当数量 = 0 时，不显示徽章

2. **筛选联动**
   - 标签统计会根据筛选条件（日期、部门、销售人员）动态更新
   - 切换标签时，列表数据会相应过滤

3. **实时更新**
   - 执行批量操作后，自动刷新统计数据
   - 点击刷新按钮，重新加载统计

## 注意事项

1. **性能优化**
   - 使用 `count()` 方法统计数量，避免查询全部数据
   - 无需代收的统计需要查询订单详情，已优化查询字段

2. **数据一致性**
   - 标签统计与列表数据保持一致
   - 筛选条件对统计和列表同时生效

3. **用户体验**
   - 数字为0时不显示徽章，保持界面简洁
   - 颜色区分明显，一目了然
