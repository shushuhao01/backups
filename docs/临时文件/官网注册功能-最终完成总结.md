- 支付使用模拟模式

### 生产环境

#### 1. 配置短信服务

**方式A：数据库配置（推荐）**
```sql
INSERT INTO system_config (config_key, config_value) VALUES (
  'sms_config',
  '{
    "enabled": true,
    "accessKeyId": "你的AccessKeyId",
    "accessKeySecret": "你的AccessKeySecret",
    "signName": "你的签名",
    "templateCode": "你的模板CODE"
  }'
);
```

**方式B：环境变量**
```bash
ALIYUN_SMS_ACCESS_KEY_ID=你的AccessKeyId
ALIYUN_SMS_ACCESS_KEY_SECRET=你的AccessKeySecret
ALIYUN_SMS_SIGN_NAME=你的签名
ALIYUN_SMS_TEMPLATE_CODE=你的模板CODE
```

#### 2. 配置支付服务

**微信支付**:
```sql
INSERT INTO payment_configs (pay_type, config_data, notify_url, enabled) VALUES (
  'wechat',
  '加密后的配置数据',
  'https://你的域名/api/v1/admin/payment/notify/wechat',
  1
);
```

**支付宝**:
```sql
INSERT INTO payment_configs (pay_type, config_data, notify_url, enabled) VALUES (
  'alipay',
  '加密后的配置数据',
  'https://你的域名/api/v1/admin/payment/notify/alipay',
  1
);
```

#### 3. 设置环境变量

```bash
NODE_ENV=production
API_BASE_URL=https://你的域名
```

#### 4. 启动服务

```bash
# 后端
cd backend
npm run build
npm start

# 官网前端
cd website
npm run build
# 部署 dist 目录到 Nginx/Apache
```
