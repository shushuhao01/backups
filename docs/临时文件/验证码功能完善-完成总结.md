## 🎯 功能特性

### 1. 验证码存储

**存储方式**: 内存存储（Map）

**数据结构**:
```typescript
interface CodeData {
  code: string          // 验证码
  phone: string         // 手机号
  createdAt: number     // 创建时间
  expiresAt: number     // 过期时间
}
```

**优点**:
- 简单快速，无需额外依赖
- 适合开发和小规模部署

**生产环境建议**:
- 使用Redis存储
- 支持分布式部署
- 更好的性能和可靠性

### 2. 防刷机制

**限制规则**:
- 同一手机号60秒内只能发送1次
- 超过限制返回剩余等待时间

**实现**:
```typescript
const timeSinceLastSend = Date.now() - codeData.createdAt

if (timeSinceLastSend < this.SEND_INTERVAL) {
  const remainingSeconds = Math.ceil((this.SEND_INTERVAL - timeSinceLastSend) / 1000)
  return {
    canSend: false,
    message: `请${remainingSeconds}秒后再试`,
    remainingSeconds
  }
}
```

### 3. 自动过期

**有效期**: 5分钟

**实现方式**:
1. 保存时设置过期时间
2. 验证时检查是否过期
3. 定时任务清理过期验证码（每分钟）

```typescript
// 保存时设置过期时间
const codeData: CodeData = {
  code,
  phone,
  createdAt: now,
  expiresAt: now + this.CODE_EXPIRY  // 5分钟后过期
}

// 自动清理
setTimeout(() => {
  this.codeStore.delete(phone)
}, this.CODE_EXPIRY)
```

### 4. 防止重复使用

**机制**: 验证成功后立即删除验证码

```typescript
// 验证成功，删除验证码（防止重复使用）
this.codeStore.delete(phone)
```

## 🚀 扩展建议

### 1. 使用Redis存储（推荐生产环境）

**安装Redis客户端**:
```bash
npm install ioredis
```

**修改 VerificationCodeService**:
```typescript
import Redis from 'ioredis'

class VerificationCodeService {
  private redis: Redis

  constructor() {
    this.redis = new Redis({
      host: process.env.REDIS_HOST || 'localhost',
      port: parseInt(process.env.REDIS_PORT || '6379'),
      password: process.env.REDIS_PASSWORD
    })
  }

  async saveCode(phone: string, code: string): Promise<void> {
    // 存储到Redis，5分钟过期
    await this.redis.setex(`sms:${phone}`, 300, code)
  }

  async verifyCode(phone: string, code: string): Promise<{
    valid: boolean
    message?: string
  }> {
    const storedCode = await this.redis.get(`sms:${phone}`)
    
    if (!storedCode) {
      return { valid: false, message: '验证码不存在或已过期' }
    }
    
    if (storedCode !== code) {
      return { valid: false, message: '验证码错误' }
    }
    
    // 验证成功，删除验证码
    await this.redis.del(`sms:${phone}`)
    
    return { valid: true }
  }
}
```
