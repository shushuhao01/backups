## 🛠️ 管理命令

### 查看所有进程
```powershell
# 在Kiro中执行
listProcesses
```

### 停止服务
```powershell
# 停止CRM前端
controlPwshProcess action=stop terminalId=6

# 停止Admin后台
controlPwshProcess action=stop terminalId=7

# 停止官网
controlPwshProcess action=stop terminalId=8

# 停止后端
controlPwshProcess action=stop terminalId=4
```

### 重启服务
```powershell
# 重启后端（触发自动重启）
echo "restart" > backend/.restart-trigger
```

## 📝 注意事项

1. **端口占用**: 确保端口 3000、5173、5174、8080 未被其他程序占用
2. **数据库连接**: 确保MySQL服务运行正常，数据库 `crm_local` 可访问
3. **环境变量**: 检查各系统的 `.env` 文件配置正确
4. **热更新**: 所有前端服务支持热更新，修改代码后自动刷新
5. **后端监控**: 后端使用nodemon，修改代码后自动重启
