# 租户创建功能修复完成总结

## 修复时间
2026-03-21

## 问题描述
用户反馈两个问题：
1. Admin后台私有客户和租户客户的新建表单中，联系人和手机号没有设置为必填
2. 创建租户时提交失败

## 问题排查

### 问题1：表单验证缺失
- **租户客户表单** (`admin/src/views/tenant-customers/List.vue`)：已有联系人和手机号必填验证 ✅
- **私有客户表单** (`admin/src/views/private-customers/List.vue`)：缺少联系人和手机号必填验证 ❌

### 问题2：租户创建失败
通过测试脚本 `backend/test-tenant-creation.js` 排查，发现数据库问题：
1. **缺少 salt 字段**：users 表缺少 `salt` 字段，导致创建管理员账号失败
2. **缺少 name 字段值**：adminAccountHelper 未插入 `name` 字段
3. **缺少 role_id 字段值**：adminAccountHelper 未插入 `role_id` 字段
