## 智能预计送达计算逻辑

```
输入参数:
- logisticsStatus: 当前物流状态
- companyCode: 快递公司代码
- shipDate: 发货时间
- latestLogisticsInfo: 最新物流信息文本
- existingEstimatedDate: 已有的预计送达时间（可选）

计算规则:
1. 已签收/异常/退回 → 不返回预计时间
2. 已有有效预计时间 → 直接返回
3. 派送中 → 当前时间 + 1天
4. 运输中 → 当前时间 + 2天
5. 已揽收 → 发货时间 + 快递公司标准时效
6. 其他 → 当前时间 + 3天（默认）

快递公司时效配置:
- 顺丰/京东: 2天
- 中通/圆通/申通/韵达/百世/极兔: 3天
- EMS/邮政: 4天
- 德邦: 3天
- 其他: 3天（默认）
```
