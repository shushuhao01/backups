```typescript
// 24小时后发送第一次提醒
WHERE po.created_at BETWEEN DATE_SUB(NOW(), INTERVAL 25 HOUR) 
                        AND DATE_SUB(NOW(), INTERVAL 24 HOUR)

// 48小时后自动关闭
WHERE po.created_at < DATE_SUB(NOW(), INTERVAL 48 HOUR)
```

### 2. 防重复发送
```typescript

  WHERE template_code = 'payment_pending'
    AND related_id = ?
    AND DATE(created_at) = ?

```
