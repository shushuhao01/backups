## 一、系统概述

微信公众号通知系统是通知模板系统的重要补充渠道,通过微信公众号向关注用户推送业务通知。

### 核心功能

✅ 用户关注管理  
✅ 租户账号绑定  
✅ 模板消息推送  
✅ 二维码场景管理  
✅ 消息发送记录  
✅ Admin后台管理界面  

### 业务流程

```
1. 用户扫码关注公众号
   ↓
2. 系统记录用户信息(openid)
   ↓
3. 用户回复"绑定"或扫描绑定二维码
   ↓
4. 输入租户账号信息完成绑定
   ↓
5. 系统发送业务通知到微信
```

## 四、核心功能实现

### 4.1 用户关注流程

```typescript
// 用户关注事件处理
async handleSubscribe(openid: string, eventData: any) {
  // 1. 保存用户信息
  const follower = await wechatFollowerRepo.create({
    openid,
    nickname: eventData.nickname,
    avatarUrl: eventData.headimgurl,
    subscribeStatus: 1,
    subscribeTime: new Date(),
    subscribeScene: eventData.scene,
    qrScene: eventData.qr_scene
  })
  
  // 2. 发送欢迎消息
  await wechatService.sendTextMessage(openid, welcomeMessage)
  
  // 3. 如果是扫码关注,自动绑定租户
  if (eventData.qr_scene) {
    const scene = await getSceneByQrScene(eventData.qr_scene)
    if (scene.sceneType === 'tenant_bind') {
      await bindTenant(openid, scene.tenantId)
    }
  }
}
```

### 4.2 租户绑定流程

**方式1: 扫码绑定**
```typescript
// 生成租户专属二维码
async generateTenantBindQrcode(tenantId: string) {
  const scene = await wechatQrcodeSceneRepo.create({
    sceneType: 'tenant_bind',
    tenantId,
    expireSeconds: 2592000 // 30天
  })
  
  const qrcode = await wechatAPI.createQrcode({
    action_name: 'QR_STR_SCENE',
    action_info: {
      scene: { scene_str: `BIND_${tenantId}` }
    },
    expire_seconds: 2592000
  })
  
  scene.ticket = qrcode.ticket
  scene.qrcodeUrl = qrcode.url
  await wechatQrcodeSceneRepo.save(scene)
  
  return qrcode.url
}
```

### 4.3 消息推送流程

```typescript
// 发送模板消息
async sendTemplateMessage(
  openid: string,
  templateCode: string,
  data: any
) {
  // 1. 获取模板配置
  const template = await getTemplateConfig(templateCode)
  
  // 2. 构建消息数据
  const message = {
    touser: openid,
    template_id: template.templateId,
    url: data.url,
    data: {
      first: { value: data.title, color: '#173177' },
      keyword1: { value: data.content1 },
      keyword2: { value: data.content2 },
      remark: { value: data.remark, color: '#909399' }
    }
  }
  
  // 3. 调用微信API发送
  const result = await wechatAPI.sendTemplateMessage(message)
  
  // 4. 记录发送日志
  await wechatMessageLogRepo.create({
    openid,
    templateCode,
    sendStatus: result.errcode === 0 ? 'success' : 'failed',
    msgid: result.msgid
  })
  
  return result
}
```

## 五、配置指南

### 5.1 微信公众号配置

1. **登录微信公众平台**
   - https://mp.weixin.qq.com

2. **获取AppID和AppSecret**
   - 开发 → 基本配置
   - 复制AppID和AppSecret

3. **配置服务器**
   - 服务器URL: `https://api.yunkes.com/wechat/callback`
   - Token: 自定义(如: yunkes_crm_2024)
   - EncodingAESKey: 随机生成

4. **配置IP白名单**
   - 添加服务器IP地址

5. **申请模板消息**
   - 功能 → 模板消息
   - 添加所需模板

2. **填写基本信息**
   - AppID
   - AppSecret
   - Token
   - EncodingAESKey

3. **配置自动回复**
   - 关注后欢迎语

## 九、常见问题

### Q1: 如何获取用户openid?
A: 用户关注公众号时,微信会推送关注事件,包含openid。

### Q2: 如何实现租户绑定?
A: 两种方式:
1. 扫描租户专属二维码自动绑定
2. 回复关键词手动输入账号密码绑定

### Q3: 模板消息有什么限制?
A: 
- 需要用户关注公众号
- 需要在微信公众平台申请模板
- 每月有发送次数限制

### Q4: 如何处理用户取消关注?
A: 
- 更新subscribe_status为0
- 保留用户数据
- 停止发送消息
