# 清除缓存并重启所有服务

## 🚀 快速操作步骤

### 方法1：使用重启脚本（推荐）

1. **双击运行脚本**
   ```
   restart-all-services.bat
   ```

2. **等待服务启动**
   - 后端服务会在新窗口启动（端口3000）
   - Admin后台会在新窗口启动（端口5174）
   - 官网会在新窗口启动（端口8080）

3. **强制刷新浏览器**
   - 打开Admin后台：http://localhost:5174
   - 按 `Ctrl + Shift + R` 强制刷新
   - 打开官网：http://localhost:8080
   - 按 `Ctrl + Shift + R` 强制刷新

### 方法2：手动重启

如果脚本不工作，手动执行：

1. **关闭所有Node进程**
   ```bash
   taskkill /F /IM node.exe
   ```

2. **启动后端**
   ```bash
   cd backend
   npm run dev
   ```

3. **启动Admin后台**（新终端）
   ```bash
   cd admin
   npm run dev
   ```

4. **启动官网**（新终端）
   ```bash
   cd website
   npm run dev
   ```

## ⚠️ 如果数据还是不对

### 清除浏览器缓存

**Chrome/Edge:**
1. 按 `F12` 打开开发者工具
2. 右键点击刷新按钮
3. 选择"清空缓存并硬性重新加载"

**或者:**
1. 按 `Ctrl + Shift + Delete`
2. 选择"缓存的图片和文件"
3. 点击"清除数据"

### 清除Vite缓存

```bash
# Admin后台
cd admin
rm -rf node_modules/.vite
rm -rf dist

# 官网
cd website
rm -rf node_modules/.vite
rm -rf dist
```
